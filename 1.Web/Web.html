Dasturchilar uchun veb-texnologiyalar

Ochiq veb dasturchilar uchun ajoyib imkoniyatlarni taqdim etadi. Ushbu texnologiyalardan to'liq foydalanish uchun siz ulardan qanday foydalanishni bilishingiz kerak. Quyida bizning veb-texnologiyalarimiz hujjatlariga havolalar mavjud.

Veb-ishlab chiquvchilar uchun hujjatlar

Veb-ishlab chiquvchilar uchun qo'llanmalar

Veb-dasturchilar uchun qo'llanmalar sizga maqsadlaringiz yoki ehtiyojlaringiz uchun veb-texnologiyalardan foydalanishda yordam beradigan amaliy va ko'rsatmalar beradi.

Veb texnologiyalari bo'yicha ma'lumotnomalar

Web API'lari
Vebda ilovalar yaratish uchun foydalanishingiz mumkin bo'lgan JavaScript dasturlash API'lari.
HTML
HTML veb-hujjatlar va ilovalarni tuzish uchun asosiy qurilish bloklarini taqdim etadi.
CSS
Kaskadli uslublar jadvallari veb-hujjatlar va ilovalarning ko'rinishini tasvirlash uchun ishlatiladi.
JavaScript
JavaScript - bu vebning ona dasturlash tili. JavaScript veb-hujjatlar va ilovalarni dinamik qilish uchun ishlatiladi.
WebAssembly
WebAssembly - bu vebning kompilyatsiya qilingan kod formati. U veb-hujjatlar va ilovalarni yuqori darajadagi performans bilan ishlatish uchun ishlatiladi. WebAssembly C, C++, Rust, Swift, C#, Go va boshqalarda yozilgan dasturlarga Internetda ishlash imkonini beradi.
HTTP
HTTP - bu veb-server va veb-klientlar orasidagi kommunikatsiya protokoli. HTTP Internet orqali hujjatlar, uslublar jadvallari, skriptlar, rasmlar, videolar, shriftlar va boshqa resurslarni olish va ma'lumotlarni veb-serverlarga qaytarish uchun asosiy Internet protokolidir.
Media
Media - bu veb-hujjatlar va ilovalarda foydalaniladigan audio va video resurslarni boshqish uchun ishlatiladi. Formatlar, kodeklar, protokollar, API va veb-hujjatlar va ilovalarga video, audio va tasvir mazmunini joylashtirish va oqimlash usullari.
SVG
SVG - bu veb-hujjatlar va ilovalarda foydalaniladigan skala qilinadigan vektor grafiklari formati. Masshtabli vektor grafikasi har qanday oʻlchamga silliq oʻlchamdagi tasvirlarni yaratish imkonini beradi.
