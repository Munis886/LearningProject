Siz har doim ham bir xil turdagi barcha elementlarni bir xil tarzda uslublashni xohlamaysiz. Misol uchun, agar sizning hujjatingiz yon panelli maqola bo'lsa, siz yon paneldagi va maqoladagi paragraf elementlariga boshqa uslubni qo'llashingiz mumkin. Buni qilishning usullaridan biri HTML teglariga maxsus atributni, ya'ni class atributini qo'yishdir. Kompyuter tillarida sinf bir xil xususiyatlarga ega bo'lgan narsalar guruhi uchun haqiqatan ham keng tarqalgan so'zdir. Hisoblashda davom etar ekansiz, turli tillardagi darslarning boshqa ko'plab misollarini ko'rasiz. Ammo hozircha bu yerda html-dagi class faqat maxsus atribut bo'lib, biz aytishimiz mumkinki, masalan, ba'zi paragraflar boshqa paragraflarga qaraganda boshqacha uslubga ega bo'lishi kerak.

Xo‘sh, qanday qilib sinfni yaratish kerak?

bu classga qanday qilib nom berish va undan foydalanishni boshlash uchun siz ba’zi HTML elementlariga class atributini qo‘shishingiz

<p class="introduction">
Bu sinfga ega paragraf. </p>
<p class="followup">
Bu boshqa sinfga ega </p>

va xuddi shu nom bilan CSS qoidalar to‘plamini yaratishingiz kerak.

.introduction {
   font-size: larger;
}

.followup {
   color: dimgray;
}

va classga CSS qoidalarini qo'llash usuli nuqta bilan boshlanadigan selektorni yozishdir. Sahifada siz qoidani qo'llashni istagan bir nechta elementlar bo'lishi mumkin bo'lgan hollarda siz foydalanishingiz kerak bo'lgan narsa classlardir.

Agar siz uni qo'llash uchun faqat bitta element bo'lishini bilsangiz, class o'rniga id atributidan foydalanib buni aniqroq qilishingiz mumkin.

<div id="main-article">
    CSS kuchukchalarni kuldiradi
</div>

<div id="sidebar">
    OK, aslida emas.
</div>

Id va sinf o'rtasidagi farq shundaki, id noyob bo'lishi kerak. Sahifada ma'lum bir identifikatorga ega faqat bitta element bo'lishi mumkin. Aksincha, bir xil sinfga ega bo'lgan elementlarning soni har qanday bo'lishi mumkin. Id qiymatlari sinf qadriyatlari bilan bir xil joydan kelib chiqadi, siz ularni shunchaki tuzasiz. Lekin selektordagi nuqta oʻrniga id boʻyicha elementlarga uslubni qoʻllash uchun siz # (xesh) belgisi yoki raqam belgisi yoki shulardan birini qoʻyasiz.

#sidebar {
   float: right;
   width: 20%;
   border: 1px solid green;
}
