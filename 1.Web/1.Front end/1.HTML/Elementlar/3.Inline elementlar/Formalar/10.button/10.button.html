<button>

<button> elementi, odatda, foydalanuvchi interfeysi orqali foydalanuvchiga tugma ko'rinishida taqdim etiladi, bu erda foydalanuvchi tugmani bosishi orqali amalni bajaradi. <button> elementi, odatda, 'type' atributi bilan ishlatiladi, bu erda 'type' atributi tugmaning turini ifodalaydi, masalan, 'submit', 'reset' yoki 'button'.

Element Display
<button class="favorite styled" type="button">
  Add to favorites
</button>

Atributlar

Ushbu elementning atributlari global atributlarni o'z ichiga oladi.

autofocus
Bu mantiqiy atribut sahifa yuklanganda tugma kiritish fokusiga ega boʻlishi kerakligini bildiradi. Hujjatdagi faqat bitta element bu atributga ega bo'lishi mumkin.
Element Display
<button class="favorite styled" autofocus>
  Add to favorites
</button>
command
Bu atribut, tugma bilan bog'langan buyruqni belgilaydi. command for atributi orqali ko'rsatilgan <button> boshqaruvi tomonidan boshqariladigan elementda bajarilishi kerak bo'lgan amalni belgilaydi. Mumkin qiymatlar:
  • "show-modal"
    • Tugma modal sifatida <dialog> ni ko'rsatadi. Agar dialog allaqachon modal bo'lsa, hech qanday chora ko'rilmaydi. Bu <dialog> elementida HTMLDialogElement.showModal() usulini chaqirishning deklarativ ekvivalentidir.
Element Display
<button class="favorite styled" command="show-modal" >
  Add to favorites
</button>
