<dialog> — HTML dialog elementi.

<dialog> HTML elementi modal yoki modal bo'lmagan dialog oynasini yoki o'chiriladigan ogohlantirish, inspektor yoki pastki oyna kabi boshqa interaktiv komponentni ifodalaydi.

Atributlar

Ushbu element global atributlarni o'z ichiga oladi.

Ogohlantirish: tabindex atributini <dialog> elementida ishlatmaslik kerak. Qo'shimcha eslatmalarga qarang.

closedby

<dialog> elementini yopish uchun ishlatilishi mumkin bo'lgan foydalanuvchi harakatlarining turlarini belgilaydi. Ushbu atribut dialogni yopishning uchta usulini ajratib turadi:

  • Yoritgichli rad etuvchi foydalanuvchi harakati, bunda <dialog> foydalanuvchi uning tashqarisiga bosgan yoki bosganida yopiladi. Bu "avtomatik" davlat popoverlarining "yengil ishdan bo'shatish" xatti-harakatiga teng.
  • Ish stoli platformalarida Esc tugmasini bosish yoki mobil platformalarda “orqaga” yoki “yopish” ishorasi kabi platformaga xos foydalanuvchi harakati.
  • HTMLDialogElement.close() yoki <form> jo'natmasini chaqiruvchi bosish ishlov beruvchisi bilan <button> kabi ishlab chiquvchi tomonidan belgilangan mexanizm.

Mumkin qiymatlar:

any

Muloqot oynasi uchta usuldan birortasi yordamida o'chirilishi mumkin.

closerequest

Muloqot oynasi platformaga xos foydalanuvchi harakati yoki ishlab chiquvchi tomonidan belgilangan mexanizm bilan o‘chirib qo‘yilishi mumkin.

none

Muloqot oynasi faqat ishlab chiquvchi tomonidan belgilangan mexanizm bilan o'chirilishi mumkin.

Agar <dialog> elementida ko'rsatilgan joriy yopiq qiymat bo'lmasa, u holda

  • agar u showModal() yordamida ochilgan bo'lsa, u qiymat "closerequest" kabi harakat qiladi.
  • aks holda, qiymat "yo'q" bo'lgandek harakat qiladi.
open

Muloqot oynasi faol ekanligini va o'zaro ta'sir qilish uchun mavjudligini bildiradi. Agar ochiq atribut o'rnatilmagan bo'lsa, muloqot oynasi foydalanuvchiga ko'rinmaydi. Muloqot oynalarini ko'rsatish uchun open atributdan ko'ra .show() yoki .showModal() usulidan foydalanish tavsiya etiladi. Agar <dialog> open atribut yordamida ochilsa, u modal emas.

Eslatma: Modal bo'lmagan dialog oynalarining ochiq va yopiq holatini ochiq atribut mavjudligini almashtirish orqali almashtirish mumkin bo'lsa-da, bu yondashuv tavsiya etilmaydi. Qo'shimcha ma'lumot olish uchun ochiq bo'limiga qarang.

Tavsif

HTML <dialog< elementi modal va modal bo'lmagan dialog oynalarini yaratish uchun ishlatiladi. Modal dialog oynalari boshqa UI elementlari bilan o'zaro ta'sirni bloklaydi va sahifaning qolgan qismini inert qiladi, modal bo'lmagan dialog oynalari esa sahifaning qolgan qismi bilan o'zaro ta'sir o'tkazishga imkon beradi.

JavaScript yordamida dialoglarni boshqarish

JavaScript-dan <dialog> elementini ko'rsatish va yopish uchun foydalanish mumkin. Modal dialogni ko'rsatish uchun showModal() usulidan va modal bo'lmagan dialogni ko'rsatish uchun show() usulidan foydalanishingiz mumkin. Muloqot oynasi close() usuli yordamida yoki <dialog> elementiga joylashtirilgan <form>ni yuborishda dialog usuli yordamida yopilishi mumkin. Modal dialoglarni Esc tugmachasini bosish orqali ham yopish mumkin.

Chaqiruvchi buyruqlar yordamida modal dialoglar

Modal dialoglar <button> elementlariga o'rnatilishi mumkin bo'lgan Invoker Commands API HTML atributlari buyruqlari va buyruqlari yordamida deklarativ tarzda ochilishi va yopilishi mumkin.

Buyruqning atributi <button> elementi bosilganda yuborilishi kerak bo'lgan maxsus buyruqni o'rnatadi, commandfor esa maqsadli dialog oynasining identifikatorini o'rnatadi. Muloqot oynalari uchun yuborilishi mumkin bo'lgan buyruqlar: "ko'rsatish-modal", "yopish" va "so'rov-yopish".

Quyidagi HTML atributlarni <button> elementiga qanday qo'llashni ko'rsatadi, shuning uchun uni "my-dialog" identifikatori bilan modal <dialog> ochish uchun bosish mumkin.

Element Display
<button command="show-modal" commandfor="my-dialog">Open dialog</button>

<dialog id="my-dialog">
<p>Ushbu dialog invoker buyrug'i yordamida ochildi.</p>
<button commandfor="my-dialog" command="close">Close</button>
</dialog>

Ushbu dialog invoker buyrug'i yordamida ochildi.

Popover buyruqlari yordamida modal bo'lmagan dialoglar

Modal bo'lmagan dialoglar deklarativ tarzda ochilishi, yopilishi va o'zgartirilishi mumkin Popover API HTML atributlari popovertarget va popovertargetaction, ular <button> va <input> elementlarida aniqlanishi mumkin.

Quyidagi HTML atributlarni <button> elementiga qanday qo'llashni ko'rsatadi, shuning uchun uni "my-dialog" id bilan modal <dialog>ni ko'rsatish va yashirish uchun bosish mumkin.

Element Display
<button popovertarget="my-dialog">Open dialog</button>

<dialog id="my-dialog" popover>
<p>This dialog was opened using a popovertargetaction attribute.</p>
<button popovertarget="my-dialog" popovertargetaction="hide">Close</button>
</dialog>

This dialog was opened using a popovertargetaction attribute.

Popover API shuningdek, JavaScript-da holatni olish va o'rnatish uchun ishlatilishi mumkin bo'lgan xususiyatlarni taqdim etadi.

Muloqot oynalarini yopish

Har bir <dialog> elementi uchun yopish mexanizmini ta'minlash va bu jismoniy klaviaturaga ega bo'lmagan qurilmalarda ishlashini ta'minlash muhimdir.
