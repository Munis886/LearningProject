Sinab ko'ring

<!Doctype html>
<html lang="en">
 <head>
  <title> My notes </title>
 </head>
 <body>

 </body>
</html>

Bu ochiq brauzerning elementiga joylashtirilgan hujjatni ko'rsatadigan bo'limda qanday sarlavha qo'yish kerakligini brauzerga aytdi.

headda paydo bo'lgan yana bir narsa stillar jadvallari haqida ma'lumot, shuningdek sahifada qaysi shrift ranglardan foydalanishni,

<Style> - Stillar

<!Doctype html>
<html lang="en">
 <head>
  <style>
    p { color: green }
  </style>
 </head>

 </body>
</html>

matnni boshida qanday joylashtirishni brauzerda ishlaydigan dasturlar scriptlarini ham o'z ichiga oladi, bu veb-ilovalar test kabi ishlaydi.

<Script>

<!Doctype html>
<html lang="en">
 <head>
  <script>

  </script>
 </head>

 </body>

bu brauzerda ishlaydi veb-ilovalarning ishlashi, masalan biz ko'rgan testlar shuningdek ushbu bo'limda o'rganganlaringiz metada bo'lishi mumkin, bu sahifaning o'zi haqida boshqa ma'lumotlar, masalan qidiruv tizimi uchun kalit so'zlar, ma'lumot, masalan html tegining o'ziga ham ishora qilishi mumkin.

Siz aytishingiz mumkin, bu yerda sizning hujjatingiz kompyuter tilida emas balki HTML tilida, lekin inson gaplashadigan ingliz, fransuz, tibetcha yoki suaxilli tili kabidir. Ushbu hujjatning tarkibini tushunishga yoki talaffuz qilishga harakat qiladigan dasturlar uchun, ko'rish qobiliyati buzilgan foydalanuvchilar uchun CH 80 Ingliz tilida chat yoki fransuzcha Shah talaffuz qilinadi, hozirda yagona element sarlavha tegidir chunki u HTML hujjatlarida talab qilinadi, qolganlarini keyinroq veb-sayt haqida ko'proq o'rganish orqali bilib olasiz.
