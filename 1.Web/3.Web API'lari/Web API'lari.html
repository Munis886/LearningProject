Web API'lari

Veb uchun kod yozishda juda ko'p sonli veb-API mavjud. Quyida veb-ilovangiz yoki saytingizni ishlab chiqishda foydalanishingiz mumkin bo'lgan barcha API va interfeyslar (obyekt turlari) ro'yxati keltirilgan.

Veb API'lari odatda JavaScript bilan ishlatiladi, garchi bu har doim ham shunday bo'lishi shart emas.
