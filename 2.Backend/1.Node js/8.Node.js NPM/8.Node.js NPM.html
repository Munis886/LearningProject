Node.js NPM

NPM nima?

NPM - bu Node.js paketlari yoki agar xohlasangiz modullar uchun paket menejeri.

www.npmjs.com yuklab olish va foydalanish uchun minglab bepul paketlarga ega.

Node.js-ni o'rnatganingizda NPM dasturi kompyuteringizga o'rnatiladi

NPM allaqachon kompyuteringizda ishlashga tayyor!

Paket nima?

Node.js paketi modul uchun zarur bo'lgan barcha fayllarni o'z ichiga oladi.

Modullar loyihangizga kiritishingiz mumkin bo'lgan JavaScript kutubxonalaridir.

Paketni yuklab oling

Paketni yuklab olish juda oson.

Buyruqlar qatori interfeysini oching va NPMga kerakli paketni yuklab olishni ayting.

Men "katta harf" deb nomlangan paketni yuklab olmoqchiman:

"Katta harf" ni yuklab oling: 

                            C:\Users\Your Name>npm install upper-case

Endi siz birinchi paketingizni yuklab oldingiz va o'rnatdingiz!

NPM paket joylashtiriladigan "node_modules" nomli papkani yaratadi. Kelajakda o'rnatadigan barcha paketlar ushbu jildga joylashtiriladi.

Mening loyiham endi quyidagi papka tuzilishiga ega:

C:\Users\My Name\node_modules\upper-case

Paketdan foydalanish

Paket o'rnatilgandan so'ng, u foydalanishga tayyor.

"Katta harf" to'plamini boshqa modullarni qo'shganingizdek qo'shing: 

                            var uc = require('upper-case');

“Salom dunyo!” chiqishini o‘zgartiradigan Node.js faylini yarating. katta harflar bilan:

Misol

Bu natijani beradi: 

                            var http = require('http');
                            var uc = require('upper-case');
                            http.createServer(function (req, res) {
                              res.writeHead(200, {'Content-Type': 'text/html'});
                              res.write(uc.upperCase("Hello World!"));
                              res.end();
                            }).listen(8080);

Yuqoridagi kodni "demo_uppercase.js" nomli faylga saqlang va faylni ishga tushiring:

demo_uppercase ishga tushirish: 

                            C:\Users\Your Name>node demo_uppercase.js

Agar siz kompyuteringizda bir xil amallarni bajargan bo'lsangiz, misoldagi kabi natijani ko'rasiz: http://localhost:8080
