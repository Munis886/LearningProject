Node.js hodisalari

Node.js hodisaga asoslangan ilovalar uchun juda mos keladi.

Node.js-dagi voqealar

Kompyuterdagi har bir harakat hodisadir. Ulanish amalga oshirilganda yoki fayl ochilganda.

Node.js’dagi obyektlar hodisalarni yoqishi mumkin, masalan, readStream obyekti faylni ochish va yopishda hodisalarni ishga tushirish:

Misol 

                            var fs = require('fs');
                            var rs = fs.createReadStream('./demofile.txt');
                            rs.on('open', function () {
                              console.log('The file is open');
                            });

Tadbirlar moduli

Node.js o‘rnatilgan “Voqealar” moduliga ega bo‘lib, unda siz o‘z voqealaringizni yaratishingiz, yoqishingiz va tinglashingiz mumkin.

O'rnatilgan Voqealar modulini kiritish uchun require() usuldan foydalaning. Bundan tashqari, barcha hodisa xususiyatlari va usullari EventEmitter obyektining namunasidir. Ushbu xususiyatlar va usullarga kirish uchun EventEmitter obyektini yarating: 

                            var events = require('events');
                            var eventEmitter = new events.EventEmitter();

EventEmitter obyekti

EventEmitter ob'ekti yordamida o'zingizning voqealaringizga hodisa ishlov beruvchilarini belgilashingiz mumkin.

Quyidagi misolda biz "qichqiriq" hodisasi ishga tushirilganda bajariladigan funksiya yaratdik.

Hodisani yoqish uchun emit()usuldan foydalaning.

Misol 

                            var events = require('events');
                            var eventEmitter = new events.EventEmitter();
                            
                            //Create an event handler:
                            var myEventHandler = function () {
                              console.log('I hear a scream!');
                            }
                            
                            //Assign the event handler to an event:
                            eventEmitter.on('scream', myEventHandler);
                            
                            //Fire the 'scream' event:
                            eventEmitter.emit('scream');
