Node.js HTTP moduli

O'rnatilgan HTTP moduli

Node.js-da HTTP deb nomlangan o'rnatilgan modul mavjud bo'lib, u Node.js-ga Hyper Text Transfer Protocol (HTTP) orqali ma'lumotlarni uzatish imkonini beradi.

HTTP modulini kiritish uchun quyidagi require()usuldan foydalaning:

var http = require('http');

Node.js veb-server sifatida

HTTP moduli server portlarini tinglaydigan va mijozga javob beradigan HTTP serverini yaratishi mumkin.

  createServer() HTTP serverini yaratish uchun quyidagi usuldan foydalaning :

Misol

var http = require('http');

//create a server object:
http.createServer(function (req, res) {
  res.write('Hello World!'); //write a response to the client
  res.end(); //end the response
}).listen(8080); //the server object listens on port 8080

Usulga o'tgan funktsiya http.createServer(), kimdir 8080 portidagi kompyuterga kirishga harakat qilganda bajariladi.

Yuqoridagi kodni "demo_http.js" nomli faylga saqlang va faylni ishga tushiring:

demo_http.js dasturini ishga tushiring:

C:\Users\Your Name>node demo_http.js

Agar siz kompyuteringizda bir xil amallarni bajargan bo'lsangiz, misoldagi kabi natijani ko'rasiz: http://localhost:8080

HTTP sarlavhasini qo'shing

Agar HTTP serverining javobi HTML sifatida ko'rsatilishi kerak bo'lsa, siz to'g'ri kontent turiga ega HTTP sarlavhasini kiritishingiz kerak:

Misol

Node.js faylida "myfirstmodule" modulidan foydalaning:

var http = require('http');
var dt = require('./myfirstmodule');

http.createServer(function (req, res) {
res.writeHead(200, {'Content-Type': 'text/html'});
res.write("The date and time are currently: " + dt.myDateTime());
res.end();
}).listen(8080);

`res.writeHead()` metodining birinchi argumenti holat kodidir, `200` esa hammasi yaxshi ekanligini bildiradi. Ikkinchi argument esa javob sarlavhalarini o‘z ichiga olgan obyekt hisoblanadi.

So'rovlar qatorini o'qing

http.createServer() funksiyasiga uzatilgan funksiya, mijozdan keladigan so'rovni obyekt sifatida (http.IncomingMessage obyekti) ifodalovchi req argumentiga ega.

Ushbu obyekt "url" nomli xususiyatga ega bo‘lib, u domen nomidan keyin keladigan URL qismida saqlanadi.

demo_http_url.js 

                                var http = require('http');
                                http.createServer(function (req, res) {
                                  res.writeHead(200, {'Content-Type': 'text/html'});
                                  res.write(req.url);
                                  res.end();
                                }).listen(8080);

Yuqoridagi kodni "demo_http_url.js" nomli faylga saqlang va faylni ishga tushiring:

demo_http_url.js dasturini ishga tushiring:

C:\Users\Your Name>node demo_http_url.js

Agar siz kompyuteringizda bir xil amallarni bajargan bo'lsangiz, ushbu ikkita manzilni ochishda ikki xil natijani ko'rishingiz kerak:

http://localhost:8080/summer

Bu natijani beradi:

/summer

http://localhost:8080/winter

Bu natijani beradi:

/winter

So'rovlar qatorini ajrating

So'rovlar qatorini URL moduli kabi o'qilishi mumkin bo'lgan qismlarga osongina ajratish uchun o'rnatilgan modullar mavjud.

Misol

So'rovlar qatorini o'qilishi mumkin bo'lgan qismlarga ajrating: 

                                var http = require('http');
                                var url = require('url');
                                
                                http.createServer(function (req, res) {
                                  res.writeHead(200, {'Content-Type': 'text/html'});
                                  var q = url.parse(req.url, true).query;
                                  var txt = q.year + " " + q.month;
                                  res.end(txt);
                                }).listen(8080);

Yuqoridagi kodni "demo_querystring.js" deb nomlangan faylga saqlang va faylni ishga tushiring:

demo_querystring.js dasturini ishga tushiring: 

                                C:\Users\Your Name>node demo_querystring.js

Manzil:

http://localhost:8080/?year=2017&month=Iyul

Bu natijani beradi: 

                                2017 July
