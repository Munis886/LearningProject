Node.js Ishni boshlash

Node.js yuklab oling

Rasmiy Node.js veb-saytida Node.js uchun o'rnatish ko'rsatmalari mavjud: https://nodejs.org

Ishni boshlash

Node.js-ni kompyuteringizga yuklab olib, o'rnatganingizdan so'ng, keling, veb-brauzerda "Salom dunyo" ni ko'rsatishga harakat qilaylik.

"myfirst.js" nomli Node.js faylini yarating va quyidagi kodni qo'shing:

Myfirst.js

var http = require('http');

http.createServer(function (req, res) {
res.writeHead(200, {'Content-Type': 'text/html'});
res.end('Hello World!');
}).listen(8080);

Faylni kompyuteringizga saqlang: C:\Users\ Your Name \myfirst.js

Kod kompyuterga “Salom dunyo!” deb yozishni aytadi. agar kimdir (masalan, veb-brauzer) 8080 portidagi kompyuteringizga kirishga harakat qilsa.

Hozircha kodni tushunishingiz shart emas. Keyinchalik tushuntiriladi.

Buyruqlar qatori interfeysi

Node.js fayllari kompyuteringizning "Buyruqlar qatori interfeysi" dasturida ishga tushirilishi kerak.

Kompyuteringizda buyruq qatori interfeysini qanday ochish operatsion tizimga bog'liq. Windows foydalanuvchilari uchun boshlash tugmasini bosing va "Buyruqning satri" ni qidiring yoki qidiruv maydoniga oddiygina "cmd" yozing.

"myfirst.js" faylini o'z ichiga olgan jildga o'ting, buyruq qatori interfeysi oynasi quyidagicha ko'rinishi kerak:

C:\Users\Your Name>_

Node.js faylini ishga tushiring

Siz yaratgan fayl har qanday harakatni amalga oshirishdan oldin Node.js tomonidan ishga tushirilishi kerak.
