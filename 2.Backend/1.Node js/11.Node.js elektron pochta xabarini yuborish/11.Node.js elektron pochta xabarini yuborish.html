Node.js elektron pochta xabarini yuborish

Nodemailer moduli

Nodemailer moduli kompyuteringizdan elektron pochta xabarlarini yuborishni osonlashtiradi.

Nodemailer moduli npm yordamida yuklab olinishi va o'rnatilishi mumkin: 

                            C:\Users\Your Name>npm install nodemailer

Nodemailer modulini yuklab olganingizdan so'ng modulni istalgan ilovaga kiritishingiz mumkin: 

                            var nodemailer = require('nodemailer');

Elektron pochta xabarini yuborish

Endi siz serveringizdan elektron pochta xabarlarini yuborishga tayyormiz.

Elektron pochta xabarini yuborish uchun tanlangan elektron pochta provayderingizdan foydalanuvchi nomi va paroldan foydalaning. Ushbu qo'llanma sizga elektron pochta xabarlarini yuborish uchun Gmail hisobingizdan qanday foydalanishni ko'rsatib beradi:

Misol 

                            var nodemailer = require('nodemailer');

                            var transporter = nodemailer.createTransport({
                              service: 'gmail',
                              auth: {
                                user: 'youremail@gmail.com',
                                pass: 'yourpassword'
                              }
                            });
                            
                            var mailOptions = {
                              from: 'youremail@gmail.com',
                              to: 'myfriend@yahoo.com',
                              subject: 'Sending Email using Node.js',
                              text: 'That was easy!'
                            };
                            
                            transporter.sendMail(mailOptions, function(error, info){
                              if (error) {
                                console.log(error);
                              } else {
                                console.log('Email sent: ' + info.response);
                              }
                            });

Va shunday! Endi sizning serveringiz elektron pochta xabarlarini yuborishi mumkin.

Bir nechta qabul qiluvchilar

Bir nechta qabul qiluvchiga xat yuborish uchun ularni vergul bilan ajratilgan mailOptions obyektining “to” xususiyatiga qo‘shing:

Misol

Bir nechta manzilga elektron pochta xabarini yuboring: 

                            var mailOptions = {
                                from: 'youremail@gmail.com',
                                to: 'myfriend@yahoo.com, myotherfriend@yahoo.com',
                                subject: 'Sending Email using Node.js',
                                text: 'That was easy!'
                              }

HTML yuboring

Elektron pochtangizga HTML formatidagi matnni yuborish uchun “matn” xususiyati o‘rniga “html” xususiyatidan foydalaning:

Misol

HTML o'z ichiga olgan elektron pochta xabarini yuboring: 

                            var mailOptions = {
                                from: 'youremail@gmail.com',
                                to: 'myfriend@yahoo.com',
                                subject: 'Sending Email using Node.js',
                                html: '<h1>Welcome</h1><p>That was easy!</p>'
                              }
