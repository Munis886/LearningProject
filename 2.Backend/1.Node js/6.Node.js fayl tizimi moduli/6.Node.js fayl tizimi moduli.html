Node.js fayl tizimi moduli

Node.js fayl serveri sifatida

Node.js ning fs (file system) moduli kompyuterning fayl tizimi bilan ishlash imkonini beradi.

Fayl tizimi (File System) modulini qo‘shish uchun require() metodidan foydalaning:

var fs = require('fs');

Fayl tizimi moduli uchun umumiy foydalanish:

  • Fayllarni o'qing
  • Fayllarni yarating
  • Fayllarni yangilash
  • Fayllarni o'chirish
  • Fayllarni qayta nomlash

Fayllarni o'qish

fs.readFile() metodi kompyuteringizdagi fayllarni o‘qish uchun ishlatiladi.

Bizda quyidagi HTML fayl (Node.js bilan bir xil papkada joylashgan) bor deb faraz qilaylik:

demofile1.html 

                            <html>
                            <body>
                            <h1>My Header</h1>
                            <p>My paragraph.</p>
                            </body>
                            </html>

HTML faylni o'qiydigan Node.js faylini yarating va tarkibni qaytaring:

Misol 

                            var http = require('http');
                            var fs = require('fs');
                            http.createServer(function (req, res) {
                              fs.readFile('demofile1.html', function(err, data) {
                                res.writeHead(200, {'Content-Type': 'text/html'});
                                res.write(data);
                                return res.end();
                              });
                            }).listen(8080);

Yuqoridagi kodni "demo_readfile.js" nomli faylga saqlang va faylni ishga tushiring:

demo_readfile.js dasturini ishga tushiring:

C:\Users\Your Name>node demo_readfile.js

Agar siz kompyuteringizda bir xil amallarni bajargan bo'lsangiz, misoldagi kabi natijani ko'rasiz: http://localhost:8080

Fayllar yaratish

Fayl tizimi modulida yangi fayllar yaratish usullari mavjud:

  • fs.appendFile()
  • fs.open()
  • fs.writeFile()

Usul fs.appendFile() belgilangan tarkibni faylga qo'shadi. Agar fayl mavjud bo'lmasa, fayl yaratiladi:

Misol

appendFile() usuli yordamida yangi fayl yarating: 

                            var fs = require('fs');

                            fs.appendFile('mynewfile1.txt', 'Hello content!', function (err) {
                              if (err) throw err;
                              console.log('Saved!');
                            });

fs.open() metodi ikkinchi argument sifatida "flag" qabul qiladi. Agar "w" (yozish) bayrog‘i berilgan bo‘lsa, ko‘rsatilgan fayl yozish uchun ochiladi. Agar fayl mavjud bo‘lmasa, bo‘sh fayl yaratiladi.

Misol

Open() usuli yordamida yangi, bo'sh fayl yarating: 

                                var fs = require('fs');

                                fs.open('mynewfile2.txt', 'w', function (err, file) {
                                   if (err) throw err;
                                   console.log('Saved!');
                                });

Usul fs.writeFile() belgilangan fayl va agar mavjud bo'lsa, tarkibni almashtiradi. Agar fayl mavjud bo'lmasa, ko'rsatilgan tarkibni o'z ichiga olgan yangi fayl yaratiladi:

Misol

writeFile() usuli yordamida yangi fayl yarating: 

                                var fs = require('fs');

                                fs.writeFile('mynewfile3.txt', 'Hello content!', function (err) {
                                   if (err) throw err;
                                   console.log('Saved!');
                                });

Fayllarni yangilash

Fayl tizimi modulida fayllarni yangilash usullari mavjud:

  • fs.appendFile()
  • fs.writeFile()

Usul fs.appendFile() belgilangan faylning oxiriga ko'rsatilgan tarkibni qo'shadi:

Misol

"Bu mening matnim" qo'shing. "mynewfile1.txt" faylining oxirigacha: 

                            var fs = require('fs');

                            fs.appendFile('mynewfile1.txt', ' This is my text.', function (err) {
                               if (err) throw err;
                               console.log('Updated!');
                            });

Usul fs.writeFile() belgilangan fayl va tarkibni almashtiradi:

Misol

"mynewfile3.txt" faylining mazmunini almashtiring: 

                            var fs = require('fs');

                            fs.writeFile('mynewfile3.txt', 'This is my text', function (err) {
                               if (err) throw err;
                               console.log('Replaced!');
                            });

Fayllarni o'chirish

Fayl tizimi moduli bilan faylni o'chirish uchun fs.unlink() usuldan foydalaning.

Usul fs.unlink() belgilangan faylni o'chiradi:

Misol

"mynewfile2.txt" faylini o'chirib tashlang: 

                                var fs = require('fs');

                                fs.unlink('mynewfile2.txt', function (err) {
                                   if (err) throw err;
                                   console.log('File deleted!');
                                });

Fayllarni qayta nomlash

Fayl tizimi moduli bilan fayl nomini o'zgartirish uchun fs.rename() usuldan foydalaning.

Usul fs.rename() belgilangan fayl nomini o'zgartiradi:

Misol

"mynewfile1.txt" nomini "myrenamedfile.txt" qilib o'zgartiring: 

                                var fs = require('fs');

                                fs.rename('mynewfile1.txt', 'myrenamedfile.txt', function (err) {
                                   if (err) throw err;
                                   console.log('File Renamed!');
                                });
