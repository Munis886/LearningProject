Node.js modullari

Node.js-dagi modul nima?

Modullarni JavaScript kutubxonalari bilan bir xil deb hisoblang.

Ilovangizga kiritmoqchi bo'lgan funktsiyalar to'plami.

O'rnatilgan modullar

Node.js o'rnatilgan modullar to'plamiga ega bo'lib, ulardan boshqa o'rnatmasdan foydalanishingiz mumkin.

Modullarning toʻliq roʻyxati uchun oʻrnatilgan modullar maʼlumotnomamizga qarang .

Modullarni kiriting

Modulni kiritish uchun require() modul nomi bilan funksiyadan foydalaning:

var http = require('http');

Endi sizning ilovangiz HTTP moduliga kirish huquqiga ega va server yaratishi mumkin:

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.end('Hello World!');
}).listen(8080);

O'z modullaringizni yarating

Siz o'zingizning modullaringizni yaratishingiz va ularni ilovalaringizga osongina kiritishingiz mumkin.

Quyidagi misol sana va vaqt obyektini qaytaradigan modul yaratadi:

Misol

Joriy sana va vaqtni qaytaradigan modul yarating:

exports.myDateTime = function () {
  return Date();
};

exports Xususiyatlar va usullarni modul faylidan tashqarida mavjud qilish uchun kalit so'zdan foydalaning .

Yuqoridagi kodni "myfirstmodule.js" deb nomlangan faylga saqlang.

O'z modulingizni qo'shing

Endi siz modulni istalgan Node.js faylingizga kiritishingiz va undan foydalanishingiz mumkin.

Misol

Node.js faylida "myfirstmodule" modulidan foydalaning:

var http = require('http');
var dt = require('./myfirstmodule');

http.createServer(function (req, res) {
res.writeHead(200, {'Content-Type': 'text/html'});
res.write("The date and time are currently: " + dt.myDateTime());
res.end();
}).listen(8080);

E'tibor bering, biz ./ modulni topish uchun foydalanamiz, ya'ni modul Node.js fayli bilan bir xil papkada joylashgan.

Yuqoridagi kodni "demo_module.js" deb nomlangan faylga saqlang va faylni ishga tushiring:

demo_module.js dasturini ishga tushiring:

C:\Users\Your Name>node demo_module.js

Agar siz kompyuteringizda bir xil amallarni bajargan bo'lsangiz, misoldagi kabi natijani ko'rasiz: http://localhost:8080
