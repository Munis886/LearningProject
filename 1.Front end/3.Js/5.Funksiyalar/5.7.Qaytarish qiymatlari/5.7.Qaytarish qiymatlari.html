Sizning funksiyalaringizdan qandaydir natija olishning bir necha usullari mavjud.

console.log()
JavaScript Console-ga qiymatni chop etish uchun foydalaning

Siz `console.log` dan xabarlarni konsolga chiqarish uchun foydalanishni bilishingiz kerak. Buni siz funksiya ichida yoki bevosita konsoldan qilishingiz mumkin, bu kursning birinchi darsida qilgan narsamiz.
Agar o‘sha darsni eslasangiz, `console.log` dan foydalanganingizda, xabaringizdan keyin "undefined" so‘zini ko‘rgan bo‘lsangiz kerak. Bu savol tug‘iladi: "undefined" nima va qayerdan paydo bo‘ladi?

undefined
Funksiya har doim qo'ng'iroq qiluvchiga qaytarilishini qaytaradi. Agar qaytarish qiymati ko'rsatilmagan bo'lsa, unda funksiya shunchaki aniqlanmagan holda qaytariladi.

Aslida, `console.log` funksiyasini chaqirganingizda, u `undefined`ni qaytaradi.
Har bir funksiya doimo chaqiruvchi tomoniga qandaydir qiymatni qaytaradi. Agar qaytariladigan qiymat aniqlanmagan bo‘lsa, funksiya avtomatik ravishda `undefined`ni qaytaradi.
Siz `console.log` ni chaqirib, unga xabarni berasiz, ya’ni xabarni argument sifatida uzatasiz va `undefined` qaytariladi. Shu bilan birga, funksiya xabarni konsolga chop etadi.
Shunday qilib, funksiyadan natija olishning ikki xil yo‘li bor:
1. `console.log` yordamida xabarlarni chop etish.
2. `return` kalit so‘zi va qiymat yordamida chaqiruvchiga qiymatni qaytarish.

Ikki muhim jihat:

console.log() return
JavaScript Console-ga qiymatni chop etish uchun foydalaning Funktsiyani ijro etishni to'xtatish va qo'ng'iroqni qaytarib berishni qaytarish uchun foydalaning

- `return` kalit so‘zi funksiyaning bajarilishini to‘xtatadi va chaqiruvchiga qiymatni qaytaradi. Agar qaytariladigan qiymat aniqlanmagan bo‘lsa, avtomatik ravishda `undefined` qaytariladi.
- `console.log` va `return` bir xil narsa emas.

Keling, buni kod orqali ko‘rib chiqamiz:

function isPrime(integer) {
  for(var x = 2; x < integer; x++){
   if (integer % x === 0) {
   console.log(integer + " is divisible by " + x);
   return false;
   }
  } return true;
}

Natija:

Ushbu misol ikkalasi o'rtasidagi asosiy farqni ko'rsatishi kerak.

Bizda bir funksiya bor, u butun sonni ikki va undan katta qiymatda tub son ekanligini tekshiradi.

Funksiya qanday ishlashini misol bilan ko‘ramiz:

Agar berilgan son tub bo‘lsa, funksiya `true` qaytaradi. Aks holda, son bo‘linuvchi birinchi sondan foydalanib xabarni konsolga chiqaradi va `false` qaytaradi.

Misollar:

- `isPrime(11)` chaqirilganda, natija `true`, chunki 11 prost son.
- `isPrime(49)` chaqirilganda, "49 is divisible by 7" konsolda chiqariladi va `false` qaytariladi, chunki 49 prost emas.

Shunday qilib, `return` chaqirilganida funksiya butunlay to‘xtaydi.

Katta xulosalar:

1. Har qanday funksiya doimo qandaydir qiymatni qaytaradi. Agar `return` mavjud bo‘lmasa, `undefined` qaytariladi.
2. `console.log` va `return` bir xil emas. `console.log` test va foydali fikr-mulohazalar uchun yaxshi, ammo bu qaytarish ifodasi emas.

qaytarish va chop etish bir xil emasligini tushunish muhimdir. JavaScript konsolida qiymatni chop etish faqat qiymatni ko‘rsatadi (siz uni tuzatish ishlari uchun ko‘rishingiz mumkin), lekin undan ko‘rsatilgan qiymatdan boshqa hech narsada foydalanib bo‘lmaydi. Shuning uchun JavaScript konsolida kodni sinab ko‘rish uchun siz faqat console.log dan foydalanishni yodda saqlashingiz kerak.

Jurnalda chop etish va qaytarish o‘rtasidagi farqni ko‘rish uchun quyidagi funksiyaning e’lon qilinishi va funksiyani chaqirtirishni JavaScript konsoliga joylashtiring:

function isThisWorking(input) {
console.log("Printing: isThisWorking was called and " + input + " was passed in as an argument.");
return "Returning: I am returning this string!";
}

isThisWorking(3);

Chop etadi: "Printing: isThisWorking was called and 3 was passed in as an argument"
Qaytaradi: "Returning: I am returning this string!"

Agar siz qaytarish qiymatini aniq belgilamasangiz, funksiya standart holat bo‘yicha aniqlanmaganni qaytaradi.

function isThisWorking(input) {
  console.log("Printing: isThisWorking was called and " + input + " was passed in as an argument.");
}

isThisWorking(3);

Chop etadi: "Printing: isThisWorking was called and 3 was passed in as an argument"
Qaytaradi: undefined
