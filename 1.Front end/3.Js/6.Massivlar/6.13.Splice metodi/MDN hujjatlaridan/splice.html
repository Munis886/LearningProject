Array.prototype.splice()

Massiv namunalari uchun `splice()` metodi massiv tarkibini o‘zgartiradi, mavjud elementlarni olib tashlaydi yoki almashtiradi va/yoki yangi elementlarni joyiga qo‘shadi.

Yangi massiv yaratish va uning segmentini o‘chirish va/yoki almashtirish, lekin asl massivni o‘zgartirmaslik uchun `toSpliced()` funksiyasidan foydalaning. Massivning bir qismini o‘zgartirmasdan olish uchun `slice()` funksiyasiga murojaat qiling.

Sinab ko'ring 

                                const months = ["Jan", "March", "April", "June"];
                                        months.splice(1, 0, "Feb");
                                        // Inserts at index 1
                                        console.log(months);
                                        // Expected output: Array ["Jan", "Feb", "March", "April", "June"]

                                        months.splice(4, 1, "May");
                                        // Replaces 1 element at index 4
                                        console.log(months);
                                        // Expected output: Array ["Jan", "Feb", "March", "April", "May"]

Sintaksis

Js

                            splice(start)
                            splice(start, deleteCount)
                            splice(start, deleteCount, item1)
                            splice(start, deleteCount, item1, item2)
                            splice(start, deleteCount, item1, item2, /* …, */ itemN)

Parametrlar

start

Nolga asoslangan indeks, massivni o‘zgartirish boshlanadigan joy, butun songa aylantiriladi.

  • Salbiy indeks massiv oxiridan orqaga hisoblanadi — agar `-array.length <= start < 0` bo‘lsa, `start + array.length` ishlatiladi.
  • Agar `start < -array.length` bo‘lsa, `0` ishlatiladi.
  • Agar `start >= array.length` bo‘lsa, hech qanday element o‘chirilmaydi, lekin metod qo‘shish funksiyasi sifatida ishlaydi va berilgan barcha elementlarni qo‘shadi.
  • Agar `start` argumenti tushirib qoldirilsa (`splice()` hech qanday argumentlarsiz chaqirilsa), hech qanday element o‘chirilmaydi. Bu `undefined` qiymatini uzatishdan farq qiladi, chunki `undefined` 0 ga aylantiriladi.
deleteCount
Butun son bo‘lib, start indeksidan boshlab massivdan nechta elementni olib tashlash kerakligini ko‘rsatadi.
Agar `deleteCount` tushirib qoldirilsa yoki uning qiymati `start` tomonidan belgilangan joydan keyingi elementlar soniga teng yoki undan katta bo‘lsa, start dan boshlab massivning oxirigacha bo‘lgan barcha elementlar o‘chiriladi. Biroq, agar siz **itemN** parametrini uzatmoqchi bo‘lsangiz, barcha elementlarni **start** dan keyin o‘chirish uchun `deleteCount` sifatida **Infinity** qiymatini uzatishingiz kerak, chunki aniq belgilangan `undefined` **0** ga aylantiriladi.
Agar `deleteCount` **0** yoki **manfiy** bo‘lsa, hech qanday element o‘chirilmaydi. Bunday holatda, kamida **bitta yangi element** ko‘rsatishingiz kerak (quyida qarang).
item1, …, itemN
Massivga qo‘shiladigan elementlar, start indeksidan boshlab.
Agar hech qanday element ko‘rsatilmasa, `splice()` faqat massivdan elementlarni o‘chirishni amalga oshiradi.

Qaytish qiymati

Massivdan o‘chirilgan elementlarni o‘z ichiga olgan massiv.

Agar faqat bitta element o‘chirilsa, **bitta elementdan iborat massiv** qaytariladi.

Agar hech qanday element o‘chirilmasa, **bo‘sh massiv** qaytariladi.
