Bilasizmi, har bir zamonaviy veb brauzer dasturchi uchun mo‘ljallangan vositalariga ega.

Dasturchi uchun mo‘ljallangan vositalardan(Developer Tools) foydalanmagan bo‘lsangiz, hech qisi yoq. "Developer Tools" panelini topish har doim ham oson kechmaydi. Shunday qilib, "Developer Tools" panelini o‘rganishingizga yordam berishi uchun ushbu qo‘llanmani yaratishga qaror qildik!

Google Chrome brauzeri

Chrome DevTools - bu "Google Chrome"ga o‘rnatilgan veb-sahifalarni yozish va tuzatish uchun mo‘ljallangan vositalar to'plamidir. "DevTools"dan saytingizdagi xatoliklarni tuzatish va test qilish uchun foydalaning. [Bu yerda "Chrome DevTools" haqida ko‘proq ma’lumot topasiz] ( https://developers.google.com/web/tools/chrome-devtools/ ).

"Chrome DevTools"ni ochish uchun istalgan sahifa elementini sichqonchaning o‘ng tugmasi bilan bosing va Inspectni tanlang yoki brauzer oynasining yuqori o‘ng burchagida Chrome sozlamalari menyusini oching va More Tools > Developer Toolsni tanlang. Shu bilan bir qatorda, klaviatura qisqartmalaridan foydalanishingiz mumkin: (opens in a new tab)

  • Command + Option + i (Mac)
  • Ctrl + Shift + i (Windows/Linux).

Mozilla Firefox brauzeri

"Firefox Developer Tools" panelidan "HTML", "CSS" va "JavaScript"ni har hil qurilmalarda tekshirsangiz va tahrirlasangiz bo‘ladi. Shuningdek, "Firefox"ning "Firefox Developer Edition" deb nomlangan talqinini yuklab olishingiz mumkin, u dasturchilar uchun mo‘ljallangan, bu talqinda "Firefox" brauzerining eng so‘nggi imkoniyatlaridan foydalanish imkoniyatiga ega bo‘lasiz. [Ushbu havoladan "Mozilla Firefox DevTools" haqida ko‘proq ma’lumot oling.] ( Https://developer.mozilla.org/en-US/docs/Tools(opens in a new tab)).

"Firefox Developer Tools"ni ochish uchun istalgan sahifa elementini sichqonchaning o‘ng tugmasi bilan bosing va Inspect Elementni tanlang yoki brauzeringiz oynasining yuqori o‘ng burchagida Firefox sozlamalari menyusini oching va Developerni tanlang. Shu bilan bir qatorda, klaviatura qisqartmalaridan foydalanishingiz mumkin: (opens in a new tab)

  • Command + Option + i (Mac)
  • Ctrl + Shift + i (Windows/Linux).

Internet Explorer brauzeri

Agar "Internet Explorer"dan foydalanayotgan bo‘lsangiz, F12 tugmachasini bosib "Developer Tools" paneliga kirishingiz mumkin. Brauzer imkoniyatlari turli xil versiyalarda farq qiladi, ammo "Internet Explorer 8"dan boshlab ancha farqlar kamaygan. Quyida har bir versiya uchun ma’lumot olishingiz uchun havolalar taqdim etganmiz, lekin "Microsoft"ning yangi talqinidagi brauzer "Microsoft Edge"ni ishlatmoqchi bo‘lsangiz, keyingi bo‘lim bilan tanishib chiqing.

  • Internet Explorer 8(opens in a new tab)
  • Internet Explorer 9(opens in a new tab)(opens in a new tab)
  • Internet Explorer 10(opens in a new tab)
  • Internet Explorer 11(opens in a new tab)

Microsoft Edge brauzeri

"Microsoft Edge" brauzeri "Internet Explorer" solishtirganda, dasturchiga yangi va zamonaviy imkoniyatlarni taqdim etadi. Dasturchi uchun yangi ish vositalari "TypeScript"da yaratilgan va uzluksiz ishlab turadi, shuning uchun ularni qayta yuklash shart emas. Bundan tashqari, F12 tugmachasi bilan yuklanadigan "Developer Tools" paneli bilan ishlaga doir ma’lumotnoma endi [GitHub]da to’liq taqdim etilgan(https://github.com/MicrosoftDocs/edge-developer(opens in a new tab)). "Internet Explorer"ga o‘xshab, "Developer Tools" panelini chaqirish uchun shunchaki F12 tugmachasini bosasiz.

Safari brauzeri

Mac operatsion tizimining har qanday foydalanuvchilari uchun Safari "Web Inspector"i paneli taqdim etiladi. Ushbu panel orqali veb-sahifalarni osonlik bilan o‘zgartirasiz, xatoliklarini to‘g‘rilaysiz va har hil platformalarda ishlashi uchun optimallashtirasiz. [Safari "Web Inspector"i to‘g‘risida ko‘proq ma’lumotni bu yerdan topishingiz mumkin(Https://developer.apple.com/safari/tools/(opens in a new tab)).

"Safari"-"ning "Web Development Tools" paneliga kirish uchun "Advanced preferences" menyusiga otasiz. Opera brauzeri

Tez va kuchli hisoblanmush Opera brauzeri, dasturchilar uchun to‘la qonli imkoniyatlar bilan birga keladi. "Opera Dragonfly" deb nomlangan bu brauzer veb ilovalar yaratishingizda ishingizni yengillashtirish uchun mo‘ljallangan. "Dragonfly Opera" haqida ko‘proq ma’lumotni bu yerdan olasiz(opens in a new tab).

"Opera Dragonfly" brauzerini quyidagi klaviatura qisqartmalari yordamida yuklashingiz mumkin:

  • Command + Option + i (Mac)
  • Ctrl + Shift + i (Windows/Linux).

Shu bilan bir qatorda, sahifada sichqonchaning o‘ng tugmachasini bosib, Inspect Elementni tanlash orqali ma’lum bir element bilan qulaylik bilan ishlashingiz mumkin.
