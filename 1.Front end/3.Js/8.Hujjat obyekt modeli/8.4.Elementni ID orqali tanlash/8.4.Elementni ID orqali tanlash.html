Elementni ID orqali tanlash

Keling, qandaydir ID atributdan foydalanib elementlarga murojaat qilish uchun JavaScript va DOMdan qanday foydalanishimiz mumkinligini ko‘rib chiqamiz.

Oldingi bo‘limdagi hujjat obyekti esingizdami? Yaxshi, endi biz undan foydalanishni boshlaymiz! Shuni esda tutingki, hujjat obyekti JavaScript kabi hamdir. Bu kalit/qiymat juftligiga ega ekanligini anglatadi. Ba’zi qiymatlar shunchaki ma’lumotlarning bir qismi, boshqalari esa funksiyalarning ayrim turini ta’minlaydigan funksiyalar (shuningdek, metodlar deb nomlanadi!) DOM ni birinchi usulini ko‘rib chiqamiz. .getElementById() metodi:

document.getElementById();

Agar biz yuqoridagi kodni konsolda ishlatsak, biz hech narsa ololmaymiz, chunki olish uchun biron bir elementning IDsini ko‘rsatmadik! Biz topmoqchi bo‘lgan element IDni .getElementByID()ga satrni uzatib, keyin uni topib bizga qaytaradi:

document.getElementById('footer');

Ushbu metod haqida ko‘proq ma’lumot olishni istasangiz hujjatlar sahifasini ko‘rib chiqing: https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementById
Keling ushbu MDN hujjati sahifasidan ushbu metodni ishlatishda foydalanmiz.

Hozirgina nimalarni bajarganimizni takrorlashimiz uchun:

  • biz ko‘rib chiqqan sahifa uchun DevToolsni ochdik
  • Console paneliga o‘tdik
  • biz document.getElementById('content'); ni konsolda ishga tushirdik

Ushbu kodni ishga tushirish, hujjat obyekti ID "kontentiga" ega bo‘lgan elementni butun daraxtga o‘xshash tuzilishni ishlashiga olib keladi.

Selecting By ID Recap

In this section, we learned how to select a DOM element by its ID:

  • .getElementById()

There are a couple of important things to keep in mind about this method:

  • it is called on the document object
  • it returns a single item

// "qo'ng'iroq" identifikatoriga ega elementni tanlang
document.getElementById('callout');
