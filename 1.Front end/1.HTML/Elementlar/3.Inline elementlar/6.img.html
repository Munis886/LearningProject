<img> - Rasmlarni qo'shish

Mushuklarning rasmlari xaritalarisiz, yoki yangi fotosuratlar, diagrammalar, va reklamalar bannerlarsiz internet nima bo'lar edi? Rasmlar Internetning juda muvaffaqqiyatli ekanligining muhim qismidir. Siz o'zingizni veb-sahifangizga rasmlarni qanday kiritish haqida savol tug'ilgandir. Bu ajablanarli emas. Buning uchun HMTL elementi mavjud. Ushbu element rasm uchun <img> va <a> element bilan bir xil ishlaydi, ammo ba'zi muhim farqlar bilan.

<a href = "http://github.com">
   Github
</a>

Havolalar uchun element href deb nomlangan bitta atributdan foydalanadi, uning qiymati siz bog'lanmoqchi bo'lgan sahifaning URL manzilidir. Ammo img elementi ikkita atributdan foydalanadi src va alt,

code:
<img src = "https://placebear.com/800/600" alt = "photo of some bears" >

display:

src manba bu sizning web sahifangizga joylashtirmoqchi bo'lgan rasmning URL manzili. alt alternativani anglatadi va tasvirning matnli tavsifi. Bu yerda paydo bo'lgan matn odatda alternativa deb nomlanadi. Endi HTML ni yozganingizda, ikkala atributni ham qo'shishingiz kerak. src atributi brauzerga tasvirni qayerdan olish kerakligini, tasvir manbai nima ekanligini anglatadi. alt atributi brauzerga qaysi rasmni tasvirlay olmasligini aytadi. Masalan, fayl serverda bo'lmasa yoki ushbu rasmni yuklab olishdan oldin foydalanuvchining tarmoq ulanishlari ishlamasa, muqobil matn paydo bo'ladi va bu mobil qurilmalar uchun odatiy holdir. Yoki foydalanuvchilar rasmlarni ko'rmasa, masalan, ular ko'r bo'lsa. Esda tutish kerak bo'lgan narsa: img elementini sizning HTML faylingizga yozib qoldirish rasm faylingizni HTML faylingizga ko'chirmaydi. Bu shunchaki brauzerga ushbu rasm faylni qayerdan topish kerakligini, uning manbasi nima ekanligini aytadi. Rasm fayli hali ham lokal disk yoki serverning biror-bir joyida joylashgan bo'lishi kerak. IMG elementi bo'sh elementdir, masalan BR kabi. U hech qanday matn tarkibiga ega emas va unga yaqin teg yo'q. Yodda tutingki, HTML elementning tarkibi p elementida bo'lgani kabi, ochiq va yopiq teglar orasidagi matn bo'ladi. Bu yerda ishlatadigan ma'noda src va alt rasm, tegi mazmunga ega emas, ular ushbu elementning HTML atributilari IMG elementi Inline (ichki) element. Siz rasmni matnning o'rtasiga qo'yishingiz mumkin. Agar rasm atrofidagi matndan uzunroq bo'lsa, unda matn atrofida bo'sh joylar hosil qiladi.

Fayllar va nisbiy "URL" manzillar

Ushbu rasm URL manzilini avvalroq sezgan bo'lishingiz mumkin. U placebear.com degan saytdan olingan, ayiqlarning ko'plab tasvirlari mavjud. Agar siz o'lchamlari 800 dan 600 pikselgacha bo'lgan rasmni xohlasangiz, placebore.com ayiqni tasviri ekanligiga amin bo'lsangiz sizga buni topib beradi.

<img src = "https://placebear.com/800/600" alt = "photo of some bears" >

Ushbu rasmni note.html faylimga qo'shish uchun, men uni <img> tegiga qo'shaman, va brauzerim ushbu HTML faylni ochganda, placebore.com saytiga kirib, ushbu rasmni, kerakligini bildiradi, va sahifaga joylashtiradi.
