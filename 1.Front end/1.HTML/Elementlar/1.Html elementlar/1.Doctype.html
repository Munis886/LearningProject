HTML parchalari. Brauzeringiz nimalarnidir ko'rsatishi mumkin, ammo ularni to'liq huquqli HTML hujjatga ya'ni, veb sahifaga aylantirish uchun yana bir nechta narsalarni qo'shishingiz mumkin. Siz shu paytgacha yozgan HTML hikoyaning o'rtasidan bir nechta jumlalarga o'xshaydi. Ammo nashr etilgan hikoyada odatda jumlalar emas, balki sarlavha va muallif kabi narsalar mavjud. Xuddi shunday barcha HTML hujjatiga to'ldirish uchun qo'shishingiz kerak bo'lgan bir nechta narsalar mavjud. Birinchidan, brauzer qanday hujjat ekanligini qayerdan biladi? Fayl nom kengaytmasi ushbu hikoyaning faqat bir qismini aytib beradi. HTML ko'p yillar davomida bo'lib kelgan, va HTML ning turli-xil versiyalari mavjud. Biz faylni eng yuqori qismiga maxsus teg qo'yib, HTML turini ishlatamiz. Bu teg Doctype tegidir. Teg biroz boshqacha tarzda yozilgan. Oldingi HTML versiyalarida ushbu Doctype, teglar juda uzoq va chalkash bo'lib qoldi.

Oxir oqibat HTML muhandislari ularni ma'lum bir soddalashtirishga qaror qilishdi. Zamonaviy HTMlda DOCTYPE tegi quyidagicha ko'rinadi. <DOCTYPE html>. Xuddi shunday. Siz yozadigan har bir HTML hujjatining yuqorisiga shunchaki joylashtirasiz. DOCTYPE tegi nima qiladi? Bu bizning brauzerimizga hujjatni zamonaviy HTML kabi, ko'rib chiqish niyyatida ekanligini aytadi. Bu oddiy HTML uchun juda muhim emas, lekin HTML murakkablashganda ko'p xatolarni oldini oladi. DOCTYPE tegisiz brauzer, quirks deb nomlangan rejimga kiradi. Qaysi HTML shakllariga ko'proq mos kelishga harakat qilmoqda? Shunday qilib hujjatlaringizga DOCTYPE tegini qo'yish to'g'ri yo'ldir.
