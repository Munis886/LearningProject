XML: Kengaytiriladigan belgilash tili

Kengaytiriladigan belgilash tili - bu Hujjat obyekti modelining qat'iy seriyalashtirilishi .

EXSLT
EXSLT - bu XML hujjatida transformatsiyalarni amalga oshirish uchun funktsiyalarni ta'minlaydigan modullarga ajratilgan XSLT kengaytmalari to'plami. EXSLT funktsiyasidan foydalanish uchun siz funksiya joylashgan nom maydonini e'lon qilishingiz va keyin funksiyani chaqirishda tegishli prefiksdan foydalanishingiz kerak.

XML qo'llanmalari
Ushbu sahifada XML uchun qo'llanmalar keltirilgan.

XPath
XPath XML Path Language degan ma'noni anglatadi. U XML hujjatining turli qismlariga moslashuvchan manzil berish (ko'rsatish) usulini ta'minlash uchun XML bo'lmagan sintaksisdan foydalanadi . Bundan tashqari, hujjat ichidagi manzillangan tugunlarni ularning naqshga mos kelishini yoki kelmasligini aniqlash uchun sinab ko'rish uchun ham foydalanish mumkin.

XSLT: Kengaytiriladigan uslublar jadvali tilini o'zgartirish
Kengaytiriladigan uslublar jadvali tilini o'zgartirish (XSLT) - bu XML hujjatlarini o'zgartirish uchun ixtisoslashtirilgan ishlov berish dasturlari bilan birgalikda ishlatiladigan XML asosidagi til.
