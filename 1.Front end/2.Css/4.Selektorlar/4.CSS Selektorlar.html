CSS selektorlari moduli CSS qoidalari to'plami qo'llaniladigan elementlarni tanlash naqshlarini aniqlaydi va ularning o'ziga xosligi bilan birga. CSS selektor moduli bizga 60 dan ortiq selektor va besh kombinatorni taqdim etadi. Boshqa modullar qo'shimcha psevdo-sinf selektorlari va psevdo-elementlarni taqdim etadi.

CSS-da selektorlar siz uslublashni xohlagan elementlarga mos kelish yoki tanlash uchun ishlatiladigan naqshlardir. Tanlagichlar JavaScript-da, shuningdek, tugunlar ro'yxati sifatida qaytish uchun DOM tugunlarini tanlashni yoqish uchun ishlatiladi.

Selektorlar, CSS yoki JavaScript-da qo'llanilishidan qat'i nazar, HTML elementlarini ularning turi, atributlari, joriy holatlari va hatto DOMdagi joylashuviga qarab maqsadli belgilash imkonini beradi. Kombinatorlar elementlarni boshqa elementlarga munosabatiga qarab tanlash imkonini berib, elementlarni tanlashda aniqroq bo'lishga imkon beradi.
