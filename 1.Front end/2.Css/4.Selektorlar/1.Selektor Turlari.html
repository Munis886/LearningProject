Selektorlarning bir nechta turli guruhlari mavjud va sizga qaysi turdagi selektor kerakligini bilish uchun to'g'ri vositani topishga yordam beradi. Ushbu maqolaning pastki maqolalarida biz selektorlarning turli guruhlarini batafsilroq ko'rib chiqamiz.

Tur, sinf va ID tanlagichlari

Tur tanlagichlari HTML elementiga mo'ljallangan, masalan <h1>:

css

h1 {

}

Class selektorlari o'z atributi uchun ma'lum bir qiymatga ega bo'lgan elementni nishonga oladi class:

css

.box {

}

ID selektorlari o'z atributi uchun ma'lum qiymatga ega bo'lgan elementni nishonga oladi id:

css

#unique {

}

Atribut selektorlari

Ushbu selektorlar guruhi elementda ma'lum bir atribut mavjudligiga qarab elementlarni tanlashning turli usullarini beradi:

css

a[title] {

}

Yoki hatto ma'lum bir qiymatga ega atribut mavjudligiga qarab tanlov qiling:

css

a[href="https://example.com"] {

}

Psevdosinflar va psevdoelementlar

Ushbu selektorlar guruhi elementning ma'lum holatlarini shakllantiradigan psevdo-sinflarni o'z ichiga oladi. Masalan, psevdo :hover-sinf elementni faqat sichqoncha ko'rsatgichi ustiga olib kelganda tanlaydi:

css

a:hover {

}

Shuningdek, u psevdoelementlarni o'z ichiga oladi, ular elementning o'zi emas, balki uning ma'lum bir qismini tanlaydi. Masalan, ::first-linehar doim element ichidagi matnning birinchi qatorini tanlaydi ( <p>quyida a) <span> birinchi formatlangan qatorga o‘ralgan va keyin tanlangandek ishlaydi.

css

p::first-line {

}

Kombinatorlar

Yakuniy selektor guruhi bizning hujjatlarimizdagi elementlarni maqsad qilish uchun boshqa selektorlarni birlashtiradi. Quyida, masalan, <article> bolalar birlashtiruvchi ( >) yordamida elementlarning bevosita bolalari bo'lgan paragraflar tanlanadi:

css

article > p {

}

Udacity

Xullas, Ushbu maqolada biz CSS selektorlarini tanishtirdik, ular sizga ma'lum HTML elementlarini maqsad qilib qo'yish imkonini beradi. Keyinchalik, tur, sinf va identifikatorni tanlash vositalarini batafsil ko'rib chiqamiz. CSS selektor brauzerga qoida qaysi elementlarga ta'sir qilishini aytadi. Aslida siz foydalanishingiz mumkin bo'lgan bir nechta turli xil selektorlar bor, ularning juda ko'pi mavjud, ammo hozir biz ulardan bir nechtasini ko'rib chiqamiz.

p { color:blue }
em { font-size:large }

Eng sodda, bu tur selektori, bu shunchaki p yoki em kabi HTML elementlarining bir turi u hatto html hujjatining butun tanasiga qo'llaniladigan qoida bo'lishi mumkin.

body { font-family:cursive }

Masalan, bu yerda hujjat turini tanlash uchun shriftni o'zgartiradigan qoidalar to'plami, ayniqsa foydalidir.

Agar web sahifada haqiqatan ham stilni qo'llamoqchi bo'lsangiz, masalan, h1 va h2 sarlavha elementlari va hattoki bitta CSS ga oid bo'lim sarlavhalarini ishlatadigan maqola yozsangiz ham selektor muayyan turdagi barcha elementlarga qo'llaniladi. Lekin baribir har bir sarlavha elementi uchun bitta CSS qoidasini yozish kerak,

h1 { font-size: 48pt;
        font family: sans-serif; }

h2 { font-size: 32pt;
        font family: sans-serif; }

Bular har biri h1,h2 elementga yo'naltirilgan va ushbu uslublarni qo'llash, bu har bir h1,h2 elementga mos keladi va shu kabi stillarni h1,h2 qo'llaydi va hokazo.
