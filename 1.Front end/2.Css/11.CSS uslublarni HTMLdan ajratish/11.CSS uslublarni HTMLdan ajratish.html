CSS uslublarni HTMLdan ajratish

HTMLda CSS stillarini qo'llash uchun brauzerga bildiradigan bir necha xil usullar mavjud. Siz ulardan birini allaqachon ko'rgansiz. Biz shunchaki CSS kodini olamiz va shunchaki uni HTML elementlari style elementiga joylashtiramiz.

<style>
#CSS {
border-radius: 5px;
background-color:darkgrey;
}
#CSS p {
padding-bottom: 0;
margin: 0;
}
</style>

Ammo bu tartibsizroq bo'lishi mumkin. Web sayt juda ko'p HTML hujjatlariga ega bo'lishi va ularning barchasida bir xil stilni qo'llamoqchi bo'lishi juda oddiy, aslida. Ushbu sahifalarning har biriga bir xil CSS kodini nusxalash katta bo'shliqni yo'qotishi mumkin. Agar biz unga o'zgartirish kiritmoqchi bo'lsak, biz barcha fayllarni o'zgartirishimiz kerak bo'lib qoladi. Shunday qilib, buni amalga oshirishning yana bir usuli bor. Biz CSSni alohida faylga yozishimiz va HTMLni ushbu faylga bog'lashimiz mumkin. Buni bajarish uchun HTMLning boshiga maxsus tegni qo'yishimiz va stil faylga tenglashtirishimiz va keyin URL orqali yoki CSS faylning fayl nomini ko'rsatamiz. Nisbiy URL manzillari qoidalarini eslaysizmi? Agar internetdagi biror-bir joyda joylashgan CSS faylidan foydalansak, biz bu yerda to'liq URL manzilini yozamiz. Ammo agar CSS fayl bizning HTML bilan bir xil katalogda bo'lsa, va deylik u style.css deb nomlangan bo'lsa, biz shunchaki bu fayl nomini shu yerga qo'yamiz. Ushbu bog'lanish elementi HTML faylining bosh elementiga kiradi. Masalan sarlavha elementidan keyin biz uni qo'yishimiz mumkin. Brauzer ushbu HTML faylni yuklaganida u avtomatik ravishda style.css ni yuklaydi va mana bu stillarni ushbu fayl elementlariga qo'llaydi.

Uslublar jadvalini(keyinchalik stylesheet deb ketiladi) "HTML" hujjatiga bo‘glash

"HTML" faylingizga staylshit bog‘lash uchun, "HTML" faylning head elementi ichiga link elementini qo'shing. Link elementi sintaksisi quyidagicha ko‘rinishda bo‘ladi:

<link rel=" stylesheet " href = " style.css " >

Agar boshqa veb-serverda joylashgan staylshitni ulansangiz, href atributida to‘liq "URL" manzildan foydalanasiz. Agar o‘zingizning "HTML" faylingiz bilan bir xil papkadagi staylshitni bog‘layotgan bo‘lsangiz, fayl nomini nisbiy "URL" manzil sifatida ishlatishingiz mumkin.

"Stylesheet"mi yoki "style element"mi - ikkalasi ham emas!

E’tibor bering, "CSS" kodingizni <style></style> elementi ichiga yozishingiz mumkin, masalan...

<style>
p{color:blue;}
</style>

Ammo ikkalasini ham bir vaqtning o‘zida qilmaganligingiz ma’qul! Boshqacha qilib aytganda, yozgan "CSS" kodlaringizni "stylesheet"ga joylashtirganingizda, style elementidan foydalanishingiz shart emas - "CSS" kodlaringizni to‘g‘ridan to‘g‘ri "stylesheet" faylga hech narsa qo‘shmasdan kiritishingiz mumkin:

p{color:blue;}

Quyidagi mashqni bajarish uchun shunday qiling, shunchaki "CSS" kodni to‘g‘ridan-to‘g‘ri style.css faylingizga kiriting (ushbu faylga "CSS" kod yozayotganingizda style elementi ishlatmang).
