"CSS" sintaksisi

CSS qoidalar to'plami ikki qismdan iborat: selektor va e'lon qilish bloki selektor bu qismdir HTML element yoki qoida qo'llaniladigan elementlar ishlashidir. E'lon qilish blokida esa odatda ushbu elementlarni qanday o'zgartirishini aytadi.
li { color:green; }
Bu yerda ro'yxatdagi barcha narsalarni yashil rangga aylantiradigan qoida to'plamidan oldingi misol bor. Bu yerda selektor li va e'lon qilishda bitta yozuv mavjud block color:green CSS qoidalari har doim shunday yozilgan.

p { width:50%; }

.weird {
  color:purple;
  font-size:large;
}
Avval selektor keladi va keyin e'lon qilish bloki atrofida bu figurali qavslar mavjud bu yerda bir nechta misol bor. Bu yerda selektor bu p demak paragrafning barcha elementlariga qo'llaniladi. Bu ikkinchi biridagi hujjat tanlangan - g'alati nuqta, ya'ni bu class atributiga teng har qanday elementga nisbatan qo'llaniladi degan ma'noni anglatadi. Figurali qavs ichidagi qism atributi e'lon qilish bloki deb ataladi va u bir yoki bir nechta e'lon qilishlardan iborat bo'lib, har bir e'lon qilish brauzerga tanlangan elementlarning uslubiga aniq o'zgartirish kiritishni buyuradi. Bu yerda e'lon qilish 50% bilan ushbu blokda unda ikkinchi e'lon qilish color:purple; font-size: large; siz shu paytgacha ushbu yozuvlarning har birida tavsifga ega nom, kenglik yoki rang yoki shrift o'lchamlari kabi, qoida ta'sir qiladigan xususiyat deb aytilgan. Unda ikki nuqta va keyin ushbu xususiyatni beradigan qiymat, keyin qoida oxirida har doim nuqta-vergul mavjud bo'ladi. Masalan ushbu qoidada rang xususiyati, binafsha rang - bu xususiyatga berilgan qiymat shuning uchun qoidaga ko'ra color:purple nuqta-vergul, shuning uchun CSS sintaksisi HTML ning sintaksisiga o'xshamaydi. Lekin bu mantiqiy to'g'ri. Chunki ular haqiqatan ham turli xil narsalarni bajarishga harakat qilmoqdalar, chunki HTML qismi elementlardan tashkil topgan daraxt tuzilishini tavsiflaydi, CSS e'lon qilishida aytilishicha, bu yerda elementning o'ziga xos turi mavjud va unga qo'llaniladigan ba'zi stillar mavjud.
