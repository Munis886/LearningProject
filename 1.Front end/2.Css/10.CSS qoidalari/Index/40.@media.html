@media

@media CSS at-qoidasi bir yoki bir nechta media so'rovlari natijasiga asoslangan uslublar jadvalining bir qismini qo'llash uchun ishlatilishi mumkin. Uning yordamida siz media so'rovi va CSS blokini hujjatga qo'llash uchun belgilaysiz, agar media so'rovi kontent ishlatilayotgan qurilmaga mos keladigan bo'lsa.

Eslatma: JavaScript-da @media yordamida yaratilgan qoidalarga CSMediaRule CSS obyekt modeli interfeysi orqali kirish mumkin.

Sinab ko'ring

Sintaksis

@media at-rule kodingizning yuqori darajasiga joylashtirilishi yoki boshqa har qanday shartli guruhga joylashtirilishi mumkin.

CSS

/* At the top level of your code */
@media screen and (min-width: 900px) {
article {
padding: 1rem 3rem;
}
}
/* Nested within another conditional at-rule */
@supports (display: flex) {
@media screen and (min-width: 900px) {
article {
display: flex;
}
}
}

Media soʻrovlar sintaksisi haqida maʼlumot olish uchun Media soʻrovlaridan foydalanish boʻlimiga qarang.

Tavsif

Media so'rovining <media-query-list> tarkibiga < media-type>s, <media-feature>s va mantiqiy operatorlar kiradi.

Media turlari

<media-type> qurilmaning umumiy toifasini tavsiflaydi. only mantiqiy operatordan foydalanilgandan tashqari, media turi ixtiyoriy all  turi nazarda tutilgan.

all
Barcha qurilmalar uchun javob beradi.
print
Chop etishni oldindan ko'rish rejimida ekranda ko'rilgan sahifalangan materiallar va hujjatlar uchun mo'ljallangan. (Ushbu formatlarga xos bo'lgan formatlash muammolari haqida ma'lumot olish uchun sahifalangan mediaga qarang.)
screen
Asosan ekranlar uchun mo'ljallangan.

Eslatma: CSS2.1 va Media Queries 3 bir nechta qo'shimcha media turlarini (tty, tv, proyeksiya, portativ, brayl, bo'rttirma va aural) aniqladi, lekin ular Media Queries 4 da eskirgan va ishlatilmasligi kerak.

Media xususiyatlari

<media feature> foydalanuvchi agenti, chiqish qurilmasi yoki muhitning o'ziga xos xususiyatlarini tavsiflaydi. Media funksiyasi ifodalari ularning mavjudligi, qiymati yoki qiymatlar diapazoni uchun sinovdan o‘tkaziladi va butunlay ixtiyoriydir. Har bir media xususiyati ifodasi qavslar bilan o'ralgan bo'lishi kerak.
