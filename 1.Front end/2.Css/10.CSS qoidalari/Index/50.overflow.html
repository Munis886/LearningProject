overflow

Overflow CSS stenografiya xususiyati kontent elementning toʻldirish qutisiga (toshib ketishi) gorizontal va/yoki vertikal yoʻnalishda mos kelmasa, kerakli harakatni oʻrnatadi.

Sinab ko'ring

Tarkibiy xususiyatlar

Bu xususiyat quyidagi CSS xususiyatlarining qisqartmasi hisoblanadi:

  • overflow-x
  • overflow-y

Sintaksis

CSS

/* Keyword values */
overflow: visible;
overflow: hidden;
overflow: clip;
overflow: scroll;
overflow: auto;
overflow: hidden visible;

/* Global values */
overflow: inherit;
overflow: initial;
overflow: revert;
overflow: revert-layer;
overflow: unset;

Overflow xususiyati bir yoki ikkita <overflow> kalit so'z qiymati sifatida belgilanadi. Agar faqat bitta kalit so'z ko'rsatilgan bo'lsa, overflow-x va overflow-y bir xil qiymatga o'rnatiladi. Agar ikkita kalit so'z ko'rsatilgan bo'lsa, birinchi qiymat gorizontal yo'nalishdagi overflow-x uchun, ikkinchisi esa vertikal yo'nalishdagi overflow-y uchun amal qiladi.

Values

visible
To‘ldiriladigan kontent kesilmagan va elementning to‘ldirish qutisidan tashqarida ko‘rinishi mumkin. Elementlar qutisi aylantirish konteyneri emas. Bu overflow xususiyatining standart qiymati.

Eslatma: Kalit so‘z qiymatining qoplamasi avtomatik uchun eski qiymat taxallusidir. Overlay yordamida aylantirish chiziqlari joy egallash o'rniga tarkibning ustiga chiziladi.

Media soʻrovlar sintaksisi haqida maʼlumot olish uchun Media soʻrovlaridan foydalanish boʻlimiga qarang.

Tavsif

Media so'rovining <media-query-list> tarkibiga < media-type>s, <media-feature>s va mantiqiy operatorlar kiradi.

Media turlari

<media-type> qurilmaning umumiy toifasini tavsiflaydi. only mantiqiy operatordan foydalanilgandan tashqari, media turi ixtiyoriy all  turi nazarda tutilgan.

all
Barcha qurilmalar uchun javob beradi.
print
Chop etishni oldindan ko'rish rejimida ekranda ko'rilgan sahifalangan materiallar va hujjatlar uchun mo'ljallangan. (Ushbu formatlarga xos bo'lgan formatlash muammolari haqida ma'lumot olish uchun sahifalangan mediaga qarang.)
screen
Asosan ekranlar uchun mo'ljallangan.

Eslatma: Kalit so‘z qiymatining qoplamasi avtomatik uchun eski qiymat taxallusidir. Overlay yordamida aylantirish chiziqlari joy egallash o'rniga tarkibning ustiga chiziladi.

Media xususiyatlari

<media feature> foydalanuvchi agenti, chiqish qurilmasi yoki muhitning o'ziga xos xususiyatlarini tavsiflaydi. Media funksiyasi ifodalari ularning mavjudligi, qiymati yoki qiymatlar diapazoni uchun sinovdan o‘tkaziladi va butunlay ixtiyoriydir. Har bir media xususiyati ifodasi qavslar bilan o'ralgan bo'lishi kerak.

prefers-reduced-motion
Foydalanuvchi sahifada kamroq harakatni afzal ko'radi. Media soʻrovlari darajasi 5-ga qoʻshilgan.
