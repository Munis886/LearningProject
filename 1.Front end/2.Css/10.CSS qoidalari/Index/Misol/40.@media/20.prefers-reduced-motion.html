prefers-reduced-motion

Ogohlantirish: Ushbu sahifaning pastki qismidagi o'rnatilgan misolda ba'zi o'quvchilar uchun muammoli bo'lishi mumkin bo'lgan o'lchov harakati mavjud. Vestibulyar harakat buzilishi bo'lgan o'quvchilar animatsiyani ko'rishdan oldin o'z qurilmalarida harakatni kamaytirish funksiyasini yoqishni xohlashlari mumkin.

Majburiy bo'lmagan harakat miqdorini minimallashtirish uchun foydalanuvchi o'z qurilmasida sozlamani yoqgan yoki yo'qligini aniqlash uchun prefers-reduced-motion li CSS media xususiyatidan foydalaniladi. Sozlama qurilmadagi brauzerga foydalanuvchi harakatga asoslangan animatsiyalarni olib tashlaydigan, kamaytiradigan yoki almashtiradigan interfeysni afzal ko'rishini bildirish uchun ishlatiladi.

Bunday animatsiyalar vestibulyar harakat buzilishi bo'lganlar uchun noqulaylik tug'dirishi mumkin. Katta ob'ektlarni masshtablash yoki panorama qilish kabi animatsiyalar vestibulyar harakat tetiklari bo'lishi mumkin.

CSS

@media (prefers-reduced-motion) {
/* styles to apply if a user's device settings are set to reduced motion */
}

Sintaksis

no-preference
Foydalanuvchi qurilmada hech qanday afzal ko'rmaganligini bildiradi. Ushbu kalit so'z qiymati noto'g'ri deb baholanadi.
reduce
Foydalanuvchi o'z qurilmasida harakatni kamaytirish uchun sozlamani yoqganligini bildiradi. Ushbu kalit so'z qiymati rost deb baholanadi.

Foydalanuvchi afzalliklari

Firefox uchun reduce so'rovi bajariladi, agar:

  • GTK/GNOME-da: Sozlamalar > Maxsus imkoniyatlar > Ko‘rish > Qisqartirilgan animatsiya yoqilgan.
    • GNOME-ning eski versiyalarida GNOME Tweaks > Umumiy yorlig'i (yoki versiyaga qarab Tashqi ko'rinish) > Animatsiyalar o'chirilgan.
    • Shu bilan bir qatorda, GTK 3 konfiguratsiya faylining [Settings] blokiga gtk-enable-animations = false ni qo‘shing.
  • Plazma/KDE-da: Tizim sozlamalari > Ish joyidagi xatti-harakatlar -> Umumiy xatti-harakat > “Animatsiya tezligi” toʻliq “Instant”ga oʻrnatiladi.
  • Windows 10 da: Sozlamalar > Kirish qulayligi > Displey > Windowsda animatsiyalarni ko‘rsatish.
  • Windows 11 da: Sozlamalar > Maxsus imkoniyatlar > Vizual effektlar > Animatsiya effektlari
  • MacOS tizimida: Tizim sozlamalari > Maxsus imkoniyatlar > Displey > Harakatni kamaytirish.
  • iOS-da: Sozlamalar > Maxsus imkoniyatlar > Harakat.
  • Android 9+ da: Sozlamalar > Maxsus imkoniyatlar > Animatsiyalarni olib tashlash.
  • Firefox-da about:config: ui.prefersReducedMotion deb nomlangan raqam afzalligini qo'shing va uning qiymatini to'liq animatsiya uchun 0 ga yoki harakatni qisqartirish afzalligini ko'rsatish uchun 1 ga o'rnating. Ushbu parametrga kiritilgan o'zgartirishlar darhol kuchga kiradi.
