font-variant

Font-variant CSS stenografiya xususiyati(shorthand property) shrift uchun barcha shrift variantlarini o'rnatish imkonini beradi.

Shuningdek, shrift stenografiyasidan foydalanib, CSS 2.1 darajasida (ya'ni oddiy yoki kichik harflar) belgilangan font-variantning <font-variant-css2> qiymatlarini o'rnatishingiz mumkin.

  • font-variant-alternates
  • font-variant-caps
  • font-variant-east-asian
  • font-variant-emoji
  • font-variant-ligatures
  • font-variant-numeric
  • font-variant-position
