-webkit-line-clamp

CSS xususiyati `line-clamp` blok tarkibini belgilangan qatordan oshib ketmasligini cheklashga imkon beradi.

Eslatma: Eski tizimlarni qo‘llab-quvvatlash uchun `-webkit-line-clamp` xususiyati faqat `display` xususiyati `-webkit-box` yoki `-webkit-inline-box` qilib o‘rnatilgan va `-webkit-box-orient` xususiyati `vertical` qilib belgilangan holatda ishlaydi. Ushbu prefiksli xususiyatlar eskirgan bo‘lsa-da, ushbu uch xususiyatning o‘zaro bog‘liqligi to‘liq belgilangan xatti-harakat bo‘lib, qo‘llab-quvvatlanishda davom etadi.

Ko‘pgina holatlarda siz ortiqcha qismini yashirish (overflow: hidden) sozlamasini o‘rnatishni ham xohlaysiz, aks holda mazmun kesilmaydi, lekin belgilangan satrlar sonidan keyin ellipsis (uch nuqta) ko‘rsatiladi.

Anchor elementlariga qo‘llanganda, qisqartirish matn oxirida emas, balki o‘rtasida ham sodir bo‘lishi mumkin.

Sintaksis

CSS

/ * Kalit so'zning qiymati * /
-webkit-line-clamp: none;

/* <integer> (<butun son>) qiymatlar */
-webkit-line-clamp: 3;
-webkit-line-clamp: 10;

/ * Global qiymatlar * /
-webkit-line-clamp: inherit;
-webkit-line-clamp: initial;
-webkit-line-clamp: revert;
-webkit-line-clamp: revert-layer;
-webkit-line-clamp: unset;

Qadriyatlar

none

   Bu qiymat kontentning siqilmasligini bildiradi.

<integer>

  Bu qiymat tarkib siqib qo'yiladigan qatorlar sonini bildiradi. 0 dan katta bo'lishi kerak.
