Xususiy xususiyatlar (--*): CSS o‘zgaruvchilari

Xususiyat nomlari -- prefiksi bilan boshlansa, masalan, --example-name, ular **maxsus xususiyatlar** bo‘lib, var() funksiyasi yordamida boshqa deklaratsiyalarda ishlatilishi mumkin bo‘lgan qiymatni o‘z ichiga oladi.

Maxsus xususiyatlar o‘zi e’lon qilingan element(lar) doirasida amal qiladi va kaskadga qatnashadi: bunday maxsus xususiyatning qiymati kaskad algoritmi tomonidan belgilangan deklaratsiyadan olinadi.

Sintaksis

CSS

--some-keyword: left;
--some-color: #0000ff;
--some-complex-value: 3px 6px rgb(20 32 54);

<declaration-value>

Ushbu qiymat bitta yoki bir nechta tokenlarning istalgan ketma-ketligiga mos keladi, agar ketma-ketlikda ruxsat etilmagan tokenlar bo‘lmasa. Bu, yaroqli deklaratsiya qiymati sifatida mavjud bo‘lishi mumkin bo‘lgan barcha narsani ifodalaydi.

Eslatma: Maxsus xususiyat nomlari katta-kichik harflarga sezgir — `--my-color` va `--My-color` alohida maxsus xususiyatlar sifatida ko‘rib chiqiladi.
