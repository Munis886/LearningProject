Yanada ko‘proq "CSS" o‘rganish

CSSda amalga oshirish mumkin bo'lgan juda ko'p narsalar mavjud. Ehtimol sizga CSS haqida o'rgatadigan eng muhim narsalardan biri bu CSS nima qila olishi mumkinligi haqida bilishingiz, buni keyingi mashqda ko'rib chiqamiz. Biz sizga CSS haqidagi ba'zi ma'lumotlarga havolani beramiz va shu tarzda ba'zi xususiyatlarni qidirasiz. Buning davomida siz professional web-dasturchilar kun bo'yi foydalanadigan ba'zi ma'lumotlar materiallaridan foydalanishga odatlanasiz va hamma xil xususiyatlar uchun 10 millionta video tayyorlashimiz shart emas. Ma'lumot uchun materiallar juda katta yoki juda ko'p ko'rinadigan bo'lsa, bu yaxshi, agar bu so'z qanday yozilganiga amin bo'lsangiz, lug'atdan so'z qidirishingizga to'g'ri keladi. Uni izlashga bir necha bor urinishingiz mumkin yoki siz o'zingizning sevimli qidiruv tizimingizdan foydalanishingiz orqali ko'proq o'rganishingiz mumkin.

"CSS" bo‘yicha ma’lumotnoma

Bu yerdan "MDN"ning "CSS" bo‘yicha ma’lumotnomasini topishingiz mumkin:

https://developer.mozilla.org/en-US/docs/Web/CSS/Reference

Ushbu CSS havolasidan barcha standart CSS xususiyatlari, psevdosinflar , psevdoelementlar , ma'lumotlar turlari , funktsional belgilar va qoidalarning alifbo indeksini ko'rib chiqish uchun foydalaning. Bundan tashqari, asosiy CSS tushunchalarini va turi bo'yicha tashkil etilgan tanlovchilar ro'yxatini ko'rib chiqishingiz mumkin. Shuningdek, qisqacha DOM-CSS / CSSOM ma'lumotnomasi ham kiritilgan.

Asosiy qoida sintaksisi

Uslub qoidasi sintaksisi

CSS

style-rule ::=
 selectors-list {
   properties-list
}

Qayerda:

CSS

selectors-list ::=
selector[:pseudo-class] [::pseudo-element]
[, selectors-list]

properties-list ::=
[property : value] [; properties-list]

Quyidagi selektorlar, psevdo-sinflar va psevdoelementlar indeksiga qarang. Har bir belgilangan qiymat uchun sintaksis har bir belgilangan xususiyat uchun belgilangan ma'lumotlar turiga bog'liq .

Uslub qoidalariga misollar

CSS

strong {
color: red;
}

div.menu-bar li:hover > ul {
display: block;
}

Selektorlar sintaksisi bilan boshlang'ich darajadagi kirish uchun CSS Selectors bo'yicha qo'llanmamizga qarang. Shuni yodda tutingki, qoida ta'rifidagi har qanday sintaksis xatosi butun qoidani bekor qiladi. Yaroqsiz qoidalar brauzer tomonidan e'tiborga olinmaydi. E'tibor bering, CSS qoidalarining ta'riflari to'liq ( ASCII ) matnga asoslangan, DOM-CSS / CSSOM (qoidalarni boshqarish tizimi) esa ob'ektga asoslangan .
