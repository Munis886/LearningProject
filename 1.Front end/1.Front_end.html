Front end

Html hujjatlari va dasturlari matnli fayllarda yoziladi. Matnli fayllar faqat belgilarni o'z ichiga oladi: harflar, raqamlar, belgilar va boshqalar.

Matn muharriri

Atom

Atom bu "Git Hub"dagi mutaxasisslar tomonidan yaratilgan mashhur muharrir. U Windows, Mac, Linux va boshqa tizimlar uchun mavjud.

VS code

Vs code bu "Microsoft" tomonidan yaratilgan yana bir mashhur muharrir. U Windows, Mac va Linux tizimlari uchun mavjud.

Diqqat kodingizni matn protsessoriga yozishga urinmang (masalan, Microsoft word, Pages, Google docs, yoki Text editga). Ushbu dasturlar matnga qo'shimcha yashirin formatlash ma'lumotlarini qo'shishi mumkin.

Izohlar

  1. Server bu shaxsiy kompyuteringizdan farqli maxsus kompyuter.
  2. Veb-sahida ko'riladigan narsalarning hammasi bir xil server saqlanishi kerak (turli xil serverlarda veb-sahifaning turli qismlariga ega bo'lolmaysiz).
  3. Veb-sahifaga kirganingizda, brauzeringiz serverga so'rov yuboradi.
  4. Veb-sahifada bir nechta veb-serverlarning manbalari (masalan, video yoki rasmlar kabi) bo'lishi mumkin.
  5. HTTP–Hyper Text Transfer Protocol ya'ni "Gipermatnni protokoli"dir.
  6. Ctrl+S tugmalarni bosish faylni ko'pchilik muharrirlarda saqlaydi
  7. Ctrl+R tugmalarini bosish brauzerni yangilaydi
  8. < > Ushbu belgilar ichidagi so'z teg deyiladi
  9. Ikkita bir-biriga mos keladigan teglardan va ularning orasidagi ba'zi matnlardan tashkil topgan butun blok element deb ataladi.
  10. <strong> awesome </strong>   Ushbu element strong element deb ataladi.
  11. Dasturchilar ko‘pincha "element" deganda "teg"ni nazarda tutadilar va aksincha ham bo‘lishi mumkin.
  12. "Sub" va "sup" Ular matnni vertikal ravishda subscript (satr osti) yoki superscript (satr usti) shaklida ko‘chiradi.
  13. Tilning grammatik qoidalari: sintaksis
