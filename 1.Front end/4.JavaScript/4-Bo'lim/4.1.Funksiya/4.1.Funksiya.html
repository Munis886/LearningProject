Funksiya

Funksiya o'zi operatorlarni guruhidir. Misol bilan o'rgatishga harakat qilaman. Masalan siz biror o'yin dasturini tuzdingiz. Keyin o'yinni foydalanuvchi tomonidan o'ynalayotgan bo'lsa o'yin ohiriga yetgan yoki aksinchaligini tekshirish uchun 100 lab qator kod yozdingiz. Keyin o'sha kodni dasturni, har joyida har xil qilib 10-15 marta yozib chiqish kerak bo'ladi. Keyin dasturni kodlari ham ko'payib ketadi. O'sha 100 lab qator kodlarni bitta funksiyaga yozib shu funksiyani (operatorlar guruhini) har joyda bitta nomlangan funksiya bilan yozib chiqsangiz bu juda yaxshi yo'l, Bu gaplarga tushunmagan bo'lishingiz mumkin lekin yaqin daqiqalar ichida bunga tushunib olasiz.

Funksiyani e'lon qilish uchun function xizmatchi so'zi mavjud.

Umumiy ko'rinishi: 

                    function funksiya_nomi(){
                        // operatorlar guruhi
                    }

Funksiyalarni e'lon qilishda harflar, raqamlar, tagchiziq va dollar belgilaridan foydalanishingiz mumkin.

Funksiyani e'lon qilib, undan foydalanish uchun funksiya nomini yozish kerak.

Endi bitta misol ko'ramiz: Sizga faqat JavaScript kodlarini keltiraman: 

                    function funksiya1() {
                        alert{
                            "Bu bizning birinchi funksiyamiz!!!"
                        }
                    }

Biz bu yerda function xizmatchi so'zi bilan funksiya e'lon qilyapmiz funksiya1 esa funksiya nomi ya'ni biz funksiyadan foydalanish uchun funksiya1(); deb yozsak bo'ladi. Keyin ochuvchi va yopuvchi bloklarni ichida alert() funksiyasi orqali xabar chiqarish kodini yozdik. Endi shu funksiyani ishlatish uchun, <<< funksiya1(); >>> ko'rinishida yozsak biz funksiyani chaqirgan bo'lamiz. Keling yaxshisi shuni HMTL va JavaScript kodlarini yozib keltiray. 

                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <meta http-equiv="X-UA-Compatible" content="ie=edge">
                            <title>Funksiya</title>
                            <script>
                                function funksiya1() {
                                 alert("Bu bizning birinchi funksiya");
                                }
                                 // funksiyaga chaqirish
                                 funksiya1();
                            </script>
                        </head>
                            <body>
                                 <button onclick="funksiya1()">Funksiyani chaqirish</button>
                            </body>
                        </html>
