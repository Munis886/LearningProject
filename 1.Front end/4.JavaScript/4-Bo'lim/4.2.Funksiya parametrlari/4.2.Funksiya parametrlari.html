Funksiya parametrlari

Funksiya parametrlari mavjuddir funksiya ichida parametrlarni e'lon qilish uchun qavslarni ichida yoziladi.

Funksiyalarni parametrlarini hoxlagancha qo'shish mumkin.

Endi bitta misol ko'ramiz. Bunda funksiyaning bitta satrli parametri mavjud bo'ladi.

HTML va JavaScript kodlar: 

                        <!DOCTYPE html>
                        <html lang="en">
                         <head>
                              <meta charset="UTF-8">
                              <meta name="viewport" content="width=device-width, initial-scale=1.0">
                              <meta http-equiv="X-UA-Compatible" content="ie=edge">
                              <title>Funksiya</title>
                              <script>
                                   function ism_chiqar(ism) {
                                    alert("Salom " + ism);
                                   }
                                   // funksiyani parametr bilan chaqirish
                                   ism_chiqar("Mo'minjon");
                              </script>
                         </head>
                             <body>
                                  Biz funksiyani yozuvli parametr bilan chaqirdik.
                             </body>
                        </html>
