Java Script haqida

Java script bu web uchun yaratilgan dasturlash tilidir. Agar web sahifalarni yaratish uchun HTML va CSS dan foydalanishni bilsangiz, JavaScript bu sizning web saytingizni dunyoga keltirish uchun zarur bo'lgan so'nggi qismdir. Brandan Eich 1995-yili JavaScriptni yaratganda, bu til web saytlarga interfaol va dinamik elementlarni qo'shishni osonlashtirish uchun yaratilgan. Ammo bugungi kunda JavaScript barcha turdagi dasturlar uchun ishlatiladi. Arduino orqali robot dasturlashdan tortib Unityda o'yin dasturlash kodlarini yozishgacha. Aslida hatto ba'zi kod yozuvchi muharrir dasturlar ham JavaScript orqali yaratilgan. Gap shundaki, JavaScriptning imkoniyatlari cheksizdir. Ushbu kursda siz JavaScript dasturlash tilining asoslari haqida bilib olasiz. Kurs davomida siz JavaScript dasturchilari tomonidan ishlatiladigan vositalar bilan tanishasiz. Va oxirida til asoslari bilan yetarli darajada tajribaga ega ekanligingizga ishonch hosil qilishni istaymiz. Shunday qilib, har bir bo'lim o'rtasida biz ilgari o'rgangan mavzularni mustahkamlaydigan muammolar to'plamiga kirishingiz mumkin. Boshlashdan oldin, JavaScript tarixi haqida ozroq gaplashish uchun tezroq sayohat qilaylik
