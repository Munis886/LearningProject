Massivlarni yaratishni boshqa yo'llari

Massivlarni boshqa yo'l bilan ham yaratish mumkin.

Bunda biz massivga necha element bilan chegaralanishini yozib qo'yishimiz kerak.

Oddiy misol bilan tushuntiraman:

var kurslar = new Array(3)
kurslar[0] = "HTML";
kurslar[1] = "CSS";
kurslar[2] = "Java";

Bu o'zi juda sust yo'ldir.

JavaScriptda massivlar dinamik. Bu degani talabga qarab massivni hajmi o'zi oshadi yoki kamayadi. Bu JavaScriptning eng zo'r imkoniyatlaridan biri.

Dinamik massiv hosil qilish:

var kurslar = new Array("HTML", "CSS", "JavaScript", "Delphi", "C++");

ko'rib turganingizdek massiv e'lon qilish va unga qiymat berish oson.

Massiv ichidagi ma'lumotlarga indeks orqali so'rov yuboriladi.

Indekslar kvadrat qavslar ichida yoziladi. Indeks doimo 0 dan boshlanadi.

kurslar nomli massiv e'lon qildik
var kurslar = new Array("HTML", "CSS", "JavaScript", "Delphi", "C++");

//endi kurs nomli o'zgaruvchi e'lon qilib kurslar massivdagi,
//0-indeksda turgan ma'lumotni berdik, ya'ni kurs = HTML ga
var kurs = kurslar[0];

// person nomli konstruktorni ichidagi changeName funksiyasi //orqali p obyektidagi ismni (name) xususiyatini o'zgartirayabmiz p.changeName("Hurrambek");

kurslar[1] = "C++"; //Bunda CSS ni o'rniga C++ ni yozdik

Yana bir misol:

var kurslar = new Array("PHP", "Ada 68", "Basic", "HTML", "CSS", "JavaScript", "Delphi", "C++");
document.write(kurslar[1]); //Natija: JavaScript
Bu kodni ishga tushirsak, bizga Ada 68 yozuvi chiqadi.

document.write(kurslar[10]);
