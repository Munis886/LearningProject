If, else shart operatorlari

Dastur tuzish mobaynida o'zgaruvchilar qiymatiga qarab u yoki bu natijani qabul qilishga to'g'ri keladi. Bu o'z navbatida dasturni tarmoqlanishiga olib keladi. Tarmoqlarning qaysi qismi bajarilishi shartlarga qarab aniqlanadi

Shart operatori: Shart operatori boshqarishni qaysi tarmoqqa uzatishni ta'minlaydi. Shart operatorining ikki xil ko'rinishi mavjud. Operatorning umumiy ko'rinishi va qisqa ko'rinishi.

Operatorning umumiy ko'rinishi: 

                    If (shart) {
                        operator1;
                    }
                    else {
                        operator2
                    }

Dunyoda qancha dastur bo'lsa shunchasida shart operatori qatnashgandir

Endi bitta misolni ko'ramiz, bunda son berliladi, sonni musbat yoki manfiyligini aniqlashni ko'ramiz: 

                    var x = 5;
                    if (x > 0) {
                          alert('Musbat son');
                    }
                    else {
                        alert ('Manfiy son');
                    }

Ma'lumot uchun alert bu ogohlantiruvchi oynani yaratadigan kod.

bo'ldi, bunda x o'zgaruvchisi berilgan va shart tekshirilyabdi. agar x, 0 dan katta bo'lsa Musbat son yozuvi chiqadi, aks holda Manfiy son yozuvi chiqadi.

Yana bitta misol, ikki son berilgan, agar birinchi son ikkinchi sondan katta bo'lsa 10 ga ko'paytirilsin, aks holda ikkinchi songa 10 qo'shilsin. 

                    var x1 = 10;
                    var x2 = 9;
                    if (x1 > x2) {
                       x1 = x1 * 10;
                       document.write(x1);
                    }
                    else {
                        x2 = x2 + 10;
                          document.write(x2);
                    }
