Ma'lumotlar toifalari

JavaScript dasturlash tilida ko'pgina ma'lumotlar toifalari mavjud: raqamlar (numbers), satrlar (strings), masssivlar va boshqalar

JavaScriptda boshqa dasturlash tillaridagi kabi, o'zgaruvchilarni nomlash kerak kerak emas.

JavaScriptda sonli o'zgaruvchilarni e'lon qilishga misol:

var num = 42; //bu raqamli num nomli o'zgaruvchi

Haqiqiy sonlarni esa nuqta bilan yoziladi, masalan 55 butun 100 dan 55 ni yozish uchun quyidagicha yozamiz:

var narx = 55.55;

Java Scriptda satrlar bittalik qo'shtirnoq ichida yoziladi, misollar:

var ism = 'Munis';

Quyidagicha qilib yozishimiz ham mumkin:

var ism = "Munis Bekchanov";

JavaScriptda mantiqiy toifa bilan ishlash uchun true (rost) false (yolg'on) xizmatchi so'zi mavjud.

var Aktiv = true;
var Aktiv = false;
