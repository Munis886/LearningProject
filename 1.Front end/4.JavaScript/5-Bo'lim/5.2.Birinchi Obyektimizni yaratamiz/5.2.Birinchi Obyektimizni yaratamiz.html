Birinchi Obyektimizni yaratamiz

Avvalgi darsda JavaScript dasturlash tilida oddiy yo'l bilan obyekt bilan ishlashni ko'rgandik. Oldingi ko'rinishda qilingan obyektlar faqat o'rganish uchun edi. Endi obyekt bilan ishlashni mustahkamlaymiz.

Hozir sizga obyektlarni yaratishni standart yo'lini ko'rsataman. Ya'ni obyekt konstruktorini yaratishni ko'ramiz:

function person(name.age, color) {
  this.name = name;
  this.age = age;
  this.color = color;
}

Tepadagi person nomli funksiya obyekt konstruktoridir.

this xizmatchi so'zi xozirgi obyektni anglatadi.

Eslatma: this bu o'zgaruvchi emas, bu bizga obyektlar konstruktorlarini yaratishda zarur bo'ladi.

Demak obyekt konstruktorini yaratishni o'rgandik. Endigi navbat new xizmatchi so'zini o'rganishga.

new xizmatchi so'zi obyektlarni o'z toifasida yaratishga imkon beradi.

Misol:
var p1 = new person("Mo'minjon", 15, "ko'k");
var p2 = new person("Anvar", 16, "qizil");
//ma'lumotlarni ekranga chiqaramiz
document.write("Birinchi shaxsning ismi: " + p1.name + "<br>");
document.write("Ikkinchi shaxsning ismi: " + p2.name + "<br>");

Demak, shaxs nomli obyektga ism, yosh, sana kabi 3 ta xususiyat kiritdik va ularga birma-bir qiymatlar berib chiqdik.

1) "=" belgisi bilan obyektga ma'lumotlar yozamiz, ma'lumotlarni vergul bilan ajratib yozarkanmiz.

2) Figurali qavslar ichiga yozish kerak ekan

Obyektlarning ichidagi ma'lumotlarga quyidagi 2 ta yo'l bilan murojaat qila olamiz. Misol bilan keltiraman

JavaScript:

var shaxs = {
ism: "Mo'minjon", yosh: 15, b: 173
};

//birinchi usul:
var x = person.yosh;

//ikkinchi usul:
var y = person['yosh'];

Obyektni ichidagi ma'lumotlarga(o'zgaruvchilarga) ikki xil yo'l bilan murojaat qilish mumkin ekan.

JavaScript dasturlash tilida ma'lumotlarni uzunligini bilish uchun length funksiyasi qo'shilgan.

Ishlatishni keltiraman:

var c = { ism: "JavaScript" };
document.write(c.ism.length);

Bu kodlar ishga tushsa bizga 10 yozuvi chiqadi.
